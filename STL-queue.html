queue

queue的定义

queue翻译为队列，在STL中主要是实现了一个先进先出的容器

queue容器内元素的访问

front

back

queue常用函数

push

  • quu.push(x)
  • 时间复杂度O(1)

front

quu.front()
返回队首元素，时间复杂度O(1)

back

quu.back()
返回队尾元素，时间复杂度O(1)

pop

quu.pop()
令队首元素出队，时间复杂度O(1)

empty

  • empty()检测queue是否为空
    • 为空则返回true
    • 非空则返回false
  • if (quu.empty() == true)
  • if (quu.empty() == false)
  • 时间复杂度O(1)

size

quu.size()
返回队内元素个数，时间复杂度O(1)

count

quu.count(1)
在队列中找1的个数

queue清空

直接用空的队列对象赋值

queue<int> q1; q1 = queue<int>();

遍历出队列

while (!Q.empty()) Q.pop();

使用swap

这种是最高效的，定义clear，保持STL容器的标准 

void clear(queue<int>& q) {
	queue<int> empty;
	swap(empty, q);
}

注意！

  • 使用front和pop函数之前必须使用empty函数判断队列是否为空，否则可能因为队空出现错误
