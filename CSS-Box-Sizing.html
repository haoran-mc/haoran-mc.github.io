CSS-BOX-SIZING

box-sizing 属性使得我们在指定元素高度和宽度时包含内边距和边框

默认的，设置一个元素高度与宽度时不会包含内边距与边框，所以元素真实的高度 = 元素的高度 + 内边距 * 2 + 边框宽度 * 2 

<div style="width: 300px; height: 100px; border: 1px solid black;">
宽度为300px，高度为100px
</div>
宽度为300px，高度为100px 
<div style="width: 300px; height: 100px; padding: 50px; border: 1px solid black;">
宽度为300px，高度为100px，但是内边距是50px
</div>
宽度为300px，高度为100px，但是内边距是50px

为了使设置高度与宽度时为总宽度与总高度， box-sizing 属性解决了这个问题 

<div style="width: 300px; height: 100px; border: 1px solid black; box-sizing: border-box;">
宽度为300px，高度为100px
</div>
宽度为300px，高度为100px 
<div style="width: 300px; height: 100px; padding: 50px; border: 1px solid black; box-sizing: border-box;">
宽度为300px，高度为100px，但是内边距是50px
</div>
宽度为300px，高度为100px，但是内边距是50px

同时给两个盒子设置了 box-sizing 属性值为 border-box ，于是两个盒子的高度与宽度都是100px、300px了，由于这种设置能更好的调整元素的大小，所以建议在所有布局中都使用 box-sizing: border-box; 

* {
  box-sizing: border-box;
}
