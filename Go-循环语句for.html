for 的三种循环方式

与多数语言不同的是，Go 语言中的循环语句只支持 for 关键字，而不支持 while 和 do-while 结构，关键字 for 的基本使用方法与 C 语言和 C++ 中非常接近：

for init; condition; post {
    // ...
}

for condition {
    // ...
}

for {
    // ...
}
init
一般为赋值表达式，给控制变量赋初值；
condition
关系表达式或逻辑表达式，循环控制条件；
post
一般为赋值表达式，给控制变量增量或减量。

for语句执行过程如下：

  1. 先对表达式 init 赋初值；
  2. 判别赋值表达式 init 是否满足给定 condition 条件，若其值为真，满足循环条件，则执行循环体内语句，然后执行 post，进入第二次循环，再判别 condition；否则判断 condition 的值为假，不满足条件，就终止for循环，执行循环体外语句。

常见的 for 循环，支持初始化语句

s := "abc"
for i, n := 0, len(s); i < n; i ++ {
    println(s[i])
}

使用循环语句时，需要注意：

  • 左花括号 {} 必须与 for 处于同一行
  • Go 语言中的 for 循环与 C 语言一样，都允许在循环条件中定义和初始化变量，惟一的区别是，Go 语言不支持以逗号为间隔的多个赋值语句，必须使用平行赋值的方式来初始化多个变量
  • Go 语言的 for 循环同样支持 continue 和 break 控制循环，但是它提供了更高级的用法： 循环控制Goto、Break、Continue

初始化语句可以被忽略，但是初始化语句之后的分号必须要写：

s := "abc"
i, n := 0, len(s)
for ; i < n; i ++ {
    println(s[i])
}

替代 while (n > 0) {}

s := "abc"
n := len(s)
for n > 0 {
    println(s[n])
    n -- 
}

替代 while (true) {}

考虑到无限循环的场景，为了让开发者不用写无聊的 for ( ; ; ) {} 和 do {} while(1) ，直接将 for 简化为：

i := 0
for {
    i ++ 
    println(i)

    if i > 10 {
        break
    }
}
