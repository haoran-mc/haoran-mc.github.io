从控制台读取输入

Java使用System.out来表示标准输出设备，用System.in来表示标准输入设备。为了完成控制台输出，只需使用println方法在控制台上显示基本数据类型值或字符串。为了获得控制台输入，可以使用Scanner类创建一个对象，以读取来自System.in的输入
Scanner input = new Scanner(System.in);
这里Scanner input声明input是一个Scanner类型的对象，语法new Scanner(System.in)表明创建了一个Scanner类型的对象，并且将它的引用值赋值给变量input，对象可以调用其方法，调用对象的方法就是让这个对象执行某个任务，可以调用nextDouble()方法来读取一个double值
double radiux = input.nextDouble();
Scanner类位于java.util包中，使用 import java.util.Scanner; 导入，如果使用Scanner类时直接使用java.til.Scanner， Scanner input = new java.util.Scanner(System.in); ，那么就可以省略import语句
print方法与println方法类似，两者不同之处在于：当显示完字符串后，println会将光标移动到下一行，而print不会
import语句有两种类型：明确导入(specific import)和通配符导入(wildcard import)
import java.util.Scanner;
import java.util.*;
除非在程序中使用某个类，否则被导入包中的类信息是不被读入的。导入语句只是告诉编译器在什么地方能找到类，声明明确导入和声明通配符导入在性能上没有差别

幂运算

使用 Math.pow(a, b); ，来计算a^b
Syatem.out.println(Math.pow(2, 3)); // Displays 8.0
Syatem.out.println(Math.pow(4, 0.5)); // Displays 2.0
Syatem.out.println(Math.pow(2.5, 2)); // Displays 6.25
Syatem.out.println(Math.pow(2.5, -2)); // Displays 0.16

数值型字面值

整型字面值

  • 为了表示一个long型的整型字面值，需要在其后加字母L(例如：2147483648L)
  • 要表示一个二进制整数字面值，在数字前使用0b或者0B
  • 要表示一个八进制整数字面值，在数字前使用0
  • 要表示一个十六进制整数字面值，在数字前使用0x或者0X
    System.out.println(0B1111); //Displays 15
    System.out.println(0777); //Displays 4095
    System.out.println(0XFFFF); //Displays 65535

浮点型字面值

默认情况下小数是double型的，在数字后加f或F表示该数为float型字面值，也可以在数字后面加d或D表示double型字面值

科学计数法

浮点型字面值也可以用a×10^b形式的科学计数法表示，例如，123.456的科学计数法形式是1.23456×10^2。一种特定的语法可以用于科学计数法的数值。例如，1.23456×10^2可以写成1.23456E2或者1.23456E+2，而1.23456×10^-2等于1.23456E-2。E(或e)表示指数，既可以用大写字母也可以用小写字母
