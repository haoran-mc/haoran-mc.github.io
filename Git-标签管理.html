GIT-标签管理

Table of Contents

  • 创建标签
  • 操作标签

创建标签

git tag <tag-name>
创建标签

标签是绑定在 commit 上的，如果忘记在某个 commit 上打一个标签，可以附加上 commit id 绑定

git tag <tag-name> <commit-id>
在确定的 commit 上绑定一个 tag
git tag
查看 tag
git show <tag-name>
查看标签信息
git tag -a <tag-name> -m "说明文字" <commit-id>
-a 指定标签名，-m 指定说明文字，创建带有说明的标签

操作标签

git tag -d <tag-name>
删除标签
git push origin <tag-name>
创建的标签都只存储在本地，不会自动推送到远程，如果要推送某个标签，需要使用这个命令
git push origin –tags
一次性推送全部尚未推送的标签

删除远程仓库的标签

git tag -d <tag-name>
先删除本地标签
git push origin :refs/tags/<tag-name>
然后从远程删除
