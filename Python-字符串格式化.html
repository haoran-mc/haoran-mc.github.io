format()格式化方法

# 默认顺序
"The fruit {} likes best is {}".format("Tom", "apple")
# 制定顺序
"The fruit {1} likes best is {0}".format("banana", "Lili")

  • 使用format方法格式化字符串时，首先需要在"{}"中输入":"引导符，之后在":"后分别设置<填充字符><对齐方式><宽度>

    设置项 可选项
    <填充字符> "*", "=", "-"……只能是一个字符，默认是空格
    <对齐方式> ^居中，<左对齐，>右对齐
    <宽度> 一个整数，指格式化后整个字符串的字符个数 
"{:*^20}".format("Mike")
# ********Mike********
"{:=<20}".format("Mike")
# Mike================
"{:.2f}".format(3.1415926)
# '3.14'
"{:=^30.4f}".format(3.1415926)
'============3.1416============'

% 操作符

print("I'm a %s. I'm %d years old." % "student", 20)
格式符 格式
%s 字符串 (采用str()的显示)
%r 字符串 (采用repr()的显示)
%c 单个字符
%b 二进制整数
%d 十进制整数
%i 十进制整数
%o 八进制整数
%x 十六进制整数
%e 指数 (基底写为e)
%E 指数 (基底写为E)
%f 浮点数
%F 浮点数，与上相同
%g 指数(e)或浮点数 (根据显示长度)
%G 指数(E)或浮点数 (根据显示长度)
