WRONG WARNING: ACCESSING NON-EXISTENT PROPERTY 'PADLEVELS' OF MODULE EXPORTS INSIDE

最近使用 neovim 的 MarkdownPreview.nvim 插件时遇到上面报错。

原因是 node 的版本太高了，我降了版本后就没报错了。

两种方法降版本：

一、安装npm插件n ,通过n模块来管理node版本

全局安装n模块
npm instlal -g n
安装当前稳定版本
n stable或者sudo n stable
安装最新版本的
n latest或者sudo n latest
安装指定版本的node
n v8.16.0
卸载指定的node版本
n rm v8.16.0

二、使用nvm管理node版本

安装nvm
brew install nvm
安装最新版本
nvm isntall node
安装指定版本
nvm install 8.16.0
查看所有版本
nvm ls
使用最新版本
nvm use node
使用指定版本
nvm use 10.16.2
