基本命令

set key01 value01
设置键值对 { key01: value01 }
get key01
获取键 key01 的值
set key01 value01 
get key01   # value01
keys *
查看所有键
set key01 value01
set key02 value02
keys *
# key01
# key02
move key01 1
将键值对移动到数据库 1 中
expire key01 10
设置 key01 这个键值对的过期时间是 10 秒
ttl key01
查看 key01 这个键值对的剩余时间
type key01
查看键 key01 对应的值的类型
exists key01
判断键 key01 是否存在
select 1
更换数据库为 1
flushdb
清除当前数据库的数据
flushall
清除所有数据库的数据
dbsize
查看数据库大小
