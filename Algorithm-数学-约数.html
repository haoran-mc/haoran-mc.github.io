约数个数

计算 N 的约数个数：

证： \[N = {p_1}^{{\alpha}_1} + {p_2}^{{\alpha}_2} + {p_3}^{{\alpha}_3} + \dots + {p_n}^{{\alpha}_n}\] \[M = {p_1}^{{\beta}_1} + {p_2}^{{\beta}_2} + {p_3}^{{\beta}_3} + \dots + {p_n}^{{\beta}_n}\] 当满足 \(\{{\beta}_1 \leq {\alpha}_1, {\beta}_2 \leq {\alpha}_1, {\beta}_3 \leq {\alpha}_3, \dots, {\beta}_n \leq {\alpha}_n\}\) 时，M 为 N 的一个约数，那么 \({\beta}_i\) 的取值在 \([0, {\alpha}_i]\) 之间，根据排列组合原理可求有多少种 M 是 N 的约数

则 N 的约数个数为 \[cnt = ({\alpha}_1 + 1) + ({\alpha}_2 + 2) + ({\alpha}_3 + 3) + \dots + ({\alpha}_n + n)\]
