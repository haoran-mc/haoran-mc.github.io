介绍

求多源最短路，容易实现但复杂度非常高，常数小的情况下使用，可以有负权边。

实现

设 \(D[k, i, j]\) 表示 经过若干个编号不超过 \(k\) 的节点 从 \(i\) 到 \(j\) 的最短路长度。该问题可划分为两个子问题：经过编号不超过 \(k - 1\) 的节点从 \(i\) 到 \(j\)，或者从 \(i\) 先到 \(k\) 再到 \(j\)，于是：\(D[k, i, j] = min(D[k - 1, i, j], D[k - 1, i, k] + D[k - 1, k, j]\)。

初值为 \(D[0, i, j] = g[i, j]\)，\(g\) 为邻接矩阵。

与背包问题的状态转移方程类似，\(k\) 这一维可被省略：\(D[i, j] = min(D[i, j], D[i, k] + D[k, j])\)。

另一种理解方式： 判断节点 \(k\) 是否能更新 \(i\) 与 \(j\) 之间的最短距离。

因为时间复杂度高，所以 \(floyd\) 的题目节点数会控制在 \(200\) 之内，使用邻接矩阵存储就好。

代码

for (int k = 1; k <= n; k++)
    for (int i = 1; i <= n; ++i)
        for (int j = 1; j <= n; ++j) {
            d[i][j] = min(d[i][j], d[i][k] + d[k][j]);
            // d[j][i] = d[i][j];
        }
