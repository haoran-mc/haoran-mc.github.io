迭代器iterator

迭代器(iterator)可以理解为一种类似指针的东西

迭代器的定义与访问

iterator -> vector

  • vector<typename>::iterator it;
    • vector<int>::iterator it;

    • vector<double>::iterator it;

      #-BEGIN_SRC C–

//二维数组的访问 vector<vector<int>>::iterator i; vector<int>::iterator j; for(i = vi.begin(); i != vi.end(); i–) { for(j = i->begin(); j != i->end(); j–) { //do something } } #-END_SRC

iterator -> string

  • string::iterator it;

    #-BEGIN_SRC C– string str = "abcde"; for (string::iterator it = str.begin(); it != str.end(); it–) printf("%c", -it);

    string::iterator it = str.begin() for (int i = 0; it-i != str.end(); i–) printf("%c", -(it - i)); #-END_SRC

iterator -> set

  • set<typename>::iterator it;
    • set<int>::iterator it;
    • set<char>::iterator it; #-BEGIN_SRC C–

set<int> st; st.insert(1); st.insert(2); st.insert(3); st.insert(4); st.insert(5); for (set<int>::iterator it = st.begin(); it != st.end(); it–) printf("%d", -it); #-END_SRC

iterator -> map

  • map<typename1, typename2>::iterator it;
    • map<char, int>::iterator it; #-BEGIN_SRC C–

map<char, int> mp; mp['a'] = 1; mp['b'] = 2; mp['c'] = 3; for (map<char, int>::iterator it = mp.begin(); it != mp.end(); it–) printf("%c %d\n", it -> first, it -> second); #-END_SRC

不使用迭代器的容器

  • queue
  • priority_queue
  • stack
  • pair
  • <algorithm>

注意！

  • 当typename是一个STL容器，那么定义时要在>>符号间加空格，防止视为移位操作
  • 除了vector和string之外的STL容器都不支持-(it - i)的访问方式
  • 除了vector和string之外的STL容器都不允许使用name.begin()-3这种迭代器加整数的写法
  • set只能使用迭代器访问
