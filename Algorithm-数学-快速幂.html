根据expo是奇数还是偶数递归求值

由于指数爆炸的原因，所以题目一般求 \(a^b % mod\)；使用快速幂可以更快的求出结果，它基于二分的思想，所以也称为二分幂：

  1. 如果b是奇数，有\(a^b = a \times a^{b - 1}\)
  2. 如果b是偶数，有\(a^b = a^{\frac{b}{2}} \times a^{\frac{b}{2}}\)

对于210：

  • \(2^{10}\)，10为偶数，\(2^{10} = 2^5 \times 2^5\)
  • \(2^5\)，5为奇数，\(2^5 = 2 \times 2^4\)
  • \(2^4\)，4为偶数，\(2^4 = 2^2 \times 2^2\)
  • \(2^2\)，2为偶数，\(2^2 = 2^1 \times 2^1\)
  • \(2^1\)，1为奇数，\(2^1 = 2 \times 2^0\)
  • 递归结束条件：\(2^0 = 1\)
long long binaryPow(long long a, long long b, long long mod) {
    //递归结束条件，递归到 2^0 = 1
    if (!b)     
        return 1;

    if (b % 2 == 1)
        return a * binaryPow(a, b - 1, mod) % mod;
    else {
        long long mul = binaryPow(a, b / 2, mod);
        return mul % mod * mul % mod;
    }
}

使用二进制方法求值

\(base^{expo}\)，将expo用二进制的方式写出来，根据二进制各个位权上是否为1来乘到结果中去

例如 \(2^5\)，510 = 01012，\(2^5 = 2^{2^2} \times 2^{2^0}\)

long long binPow(long long base, long long expo) {
    long long res = 1;
    while (expo) {
        if (expo & 1)
            res = (res * base) % mod;
        base = (base * base) % mod;
        // 也就是base分别是2, 4, 8, 16, ...
        expo >>= 1;
    }
    return res % mod;
}
