PYTHON-列表

> python语言中，将一组数据放在一对方括号"[]"中，即定义了一个列表 > 方括号中的每个数据称为元素，元素与元素之间用", "隔开，元素的个数称为列表的长度 > 列表允许多次出现相同元素

  1. 初始化列表
  2. 列表的索引
    • 正向从左向右从0递增
    • 反向从右向左从-1递减
  3. 修改元素
    • 列表名[索引] = 新值
  4. 增加元素
    • append()，列表专属，在列表尾部追加制定新元素
      • 列表名.append(新元素)
    • insert()，允许为新元素指定插入位置
      • 列表名.insert(索引，新元素)
  5. 删除元素
    • del命令
      • del 列表名[索引]
    • pop()
      • 列表名.pop(索引)，并返回该元素，默认索引为-1
    • remove()
      • 列表.remove(元素值)，直接指定待删除元素的值，如果列表中有多个相同的待删除的元素，只有索引值最小的会被删除
  6. 其他常用操作
    • len(列表)，统计和返回指定列表的长度，即列表中元素的个数
    • 运算符in和not in，用来判断指定的元素是否在列表中，如果在，返回true，否则返回false
    • 列表.index(元素)，用来在列表中查找指定元素，如果存在则返回在列表中的索引，如果存在多个指定元素，返回最小索引，如果不存在，直接报错
    • 列表.count(元素)，统计并返回列表中指定元素的个数
  7. 列表排序
    • 列表.sort()，列表元素从小到大升序排列(对中文不友好)，改变原列表中元素的顺序
    • 列表.sort(reverse = True)，列表与元素从大到小降序排列
    • sorted(列表, reverse)，生成排序后的列表副本，不改变原列表中元素的顺序
    • sorted(列表, reverse = False)，升序排列
    • sorted(列表, reverse = True)，降序排列
  8. 列表切片
    • 列表[起始索引:终止索引]
      • 缺省起始索引时，切片默认从索引0开始
      • 缺省终止索引时，切片默认到最后一个元素终止
      • 同时缺省起始索引与终止索引时，切片默认取整个列表
      • 终止索引取-1时，会取不到最后一个元素
    • 列表[起始索引:终止索引:n]
  9. 列表的扩充
    • +运算
      • list1 = list2 + list3
      • list2与list3不变，list1是list2与list3的连接
    • 列表.extend(新列表)，直接将新的列表加到原列表的后面
    • *运算，list1 * 3
  10. 列表的复制
    1. 利用切片，list2 = list1[:]
    2. list1 = list2.copy()
    3. list1 = list2
      • 这样的意思是给list2取了个新名字，改变list1，list2也会发生改变
  11. 列表的删除
    • del list1[2:4]
    • del list1[:]
    • del list1
  12. 数值列表
    • list1 = eval(input(input one list: ))
    • list2 = list(range(1, 11))
      • list3 = list(input("input one list: ")) # ['[', '1', ',', ' ', '2', ',', ' ', '3', ']']

    • 列表生成式，列表 = [循环变量相关表达式 for 循环变量 in range()]

for i int range(1, 11):
list1.append(i**2)

list2 = [i**2 for i in range(1, 11)]

    • lst.clear()
