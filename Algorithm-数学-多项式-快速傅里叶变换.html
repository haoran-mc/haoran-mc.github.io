快速傅里叶变换

快速傅里叶变换（Fast Fourier Transform，FFT）是一种可在O(nlogn) 时间内完成的离散傅里叶变换（Discrete Fourier transform，DFT）算法
在算法竞赛中的运用主要是用来加速多项式的乘法

多项式的乘法，引入傅里叶变换

对于两个多项式A(x), B(x)的乘积C(x)，假设A(x)的项数为n，其系数构成的n维向量为(a_0, a_1, a_2, … an-1)；B(x)的项数为m，其系数构成的m维向量为(b_0, b_1, b_2, … bm-1)

求C(x)的系数构成的(n + m - 1)维向量

for (int i = 0; i < n; ++i)
    for (int j = 0; j < m; ++j) 
        c[i + j] += a[i] * b[j];

关于多项式

多项式有两种表示方法，系数表达法和点值表达法

多项式由系数表示法转为点值表示法的过程，就称为DFT
相对地，把一个多项式的点值表示法转化为系数表示法的过程，就是IDFT
而FFT就是通过取某些特殊的x的点值来加速DFT和IDFT的过程

系数表达法

如果A(x)是一个n次多项式，那么所有项的系数组成的系数向量(a_0, a_1, a_2, … a_n)能够唯一确定这个多项式

点值表示法

将一组互不相同的(x_0, x_1, x_2, …, x_n)分别带入A(x)，得到取值(y_0, y_1, y_2, … y_n)

定理：一个n-1次多项式在n个不同点的取值唯一确定了该多项式
证明：假设存在不同的两个多项式A(x), B(x)同时满足对于任何一个x_i ∈ [x_0, xn-1]，有A(x_i) = B(x_i) = y_i
令C(x) = A(x) - B(x)，则C(x)也是一个n-1次多项式，那么对于所有的(x_i, y_i)，有C(x_i) = 0，即C(x)有n个根，这与代数基本定理(一个n-1次多项式在有且仅有n-1个根)相矛盾，故C(x)不是一个n-1次多项式，推倒矛盾，原命题成立

  1. 首先通过两个多项式的点值表达式得到结果的点值表达式
    已知一组横坐标(x_0, x_1, x_2, … xn-1)，得到两个多项式的纵坐标分别为 \((y_{a0}, y_{a1}, y_{a2}, ... y_{a(n-1)})，(y_{b0}, y_{b1}, y_{b2}, ... y_{b(n-1)})\)
    那么可以在O(n)的时间复杂度内得到结果的点值表达式(ya0 ⋅ yb0, ya1 ⋅ yb1, ya2 ⋅ yb2, … ya(n-1) ⋅ yb(n-1))
    其实不需要先算出两个表达式的值之后乘出结果，可以直接通过y_i = A(x_i) ⋅ B(x_i)得到

因为C(x)的项数为A(x)与B(x)的项数之和，设A(x)与B(x)的项数分别为n, m，则C(x)的项数为n + m，如果我们能够快速通过点值表达式得到系数表达式，那么我们就能在两者之间搭起一座桥了
这也是快速傅里叶变换的基本思路，由系数表达式到点值表达式到结果的点值表达式再到结果的系数表达式

关于复数

我们把形如 a + bi 这样的数叫做复数，复数集合用C来表示。其中a称为实部(real part)，b称为虚部(imaginary part) ,i为虚数单位，指满足 x^2 = -1 的一个解 \sqrt{-1} ；此外，对于这样对复数开偶次幂的数叫做虚数(imaginary number)

每一个复数a + bi都对应了一个平面上的向量(a, b)我们把这样的平面称为复平面(complex plane)，它是由水平的实轴与垂直的虚轴建立起来的复数的几何表示

故每一个复数唯一对应了一个复平面上的向量，每一个复平面上的向量也唯一对应了一个复数。其中0既被认为是实数，也被认为是虚数

其中复数z = a + bi的模长 |z| 定义为 z 在复平面的距离到原点的距离， |z| = \sqrt{a^2 + b^2}。幅角 θ 为实轴的正半轴正方向（逆时针）旋转到 z 的有向角度

由于虚数无法比较大小。复数之间的大小关系只存在等于与不等于两种关系，两个复数相等当且仅当实部虚部对应相等。对于虚部为 0 的复数之间是可以比较大小的，相当于实数之间的比较

复数之间的运算满足结合律，交换律和分配律

复数运算的加法满足平行四边形法则，乘法满足幅角相加，模长相乘

对于一个复数 z = a + bi,它的共轭复数是 z' = a - bi, z' 称为 z 的复共轭(complex conjugate)

共轭复数的性质：\(z \cdot z' = a^2 + b^2，|z| = |z'|\)

复数中的单位根

img-复平面中的单位圆

其中 \bar{OA} 单位根，表示为 eiθ ，可知 eiθ=cosθ+i⋅ sinθ
（顺便一提著名的欧拉幅角公式 eiπ+1=0 其实是由定义来的…）

严谨地，我们称x^n = 1在复数意义下的解是n次复根。显然，这样的解有n个，设ω_n = e\frac{2 \pi i}{n}，则x^n = 1的解集表示为{ωnk | k = 0, 1, … n-1}。我们称ω_n是n次单位复根(the -th root of unity)。根据复平面的知识，n次单位复根是复平面把单位圆n等分的第一个角所对应的向量。其它复根均可以用单位复根的幂表示。

将单位圆等分成 n 个部分（以单位圆与实轴正半轴的交点一个等分点），以原点为起点，圆的这 n 个 n 等分点为终点，作出 n 个向量

其中幅角为正且最小的向量称为 n 次单位向量，记为 ωn1

img-复平面中的n次向量

其余的 n-1 个向量分别为 ωn2,ωn3,……,ωnn ，它们可以由复数之间的乘法得来 ωnk = ωnk-1 ⋅ ωn1\ (2 ≤ k ≤ n)

容易看出 ωnn = ωn0 = 1

对于 ωnk ,它事实上就是 e2 π ⋅ \frac{k}{n}i

所以 ωnk = cos(2 π ⋅ \frac{k}{n}) + i ⋅ sin(2 π ⋅ \frac{k}{n})

关于单位根有两个性质

性质一(折半引理)

ω2n2k = ωnk
使用欧拉公式易证
引申 ωmnmk = ωnk

性质二(消去引理)

ωnk + \frac{n}{2} = - ωnk
由几何意义，这两者表示的向量终点是相反的，左边较右边在单位圆上多转了半圈

离散傅里叶变换DFT

归纳来说，DFT是分治地求x = ω_n^k 的时候f[x]的值，它的分治思想体现在将多项式分为奇次项和偶次项处理

首先我们单独考虑一个n项（n = 2^x）的多项式A(x)，其系数向量为(a_0, a_1, a_2, … an-1)。我们将n次单位根的0 ~ n-1次幂分别带入A(x)得到其点值向量(A(ωn0), A(ωn1), A(ωn2), …, A(ωnn-1))

这就是离散傅里叶变换（Discrete Fourier Transform）

如果朴素带入，时间复杂度也是O(n^2)的

所以我们必须要利用到单位根 ω 的特殊性质

对于 A(x) = a_0 + a_1 ⋅ x + a_2 ⋅ x^2 + … + an-1 ⋅ xn-1

考虑将其按照奇偶分组

A(x) = (a_0 + a_2 ⋅ x^2 + a_4 ⋅ x^4 + … + an-2 ⋅ xn-2) + (a_1 ⋅ x^1 + a_3 ⋅ x^3 + a_5 ⋅ x^5 + … + an-1 ⋅ xn-1)

A(x) = (a_0 + a_2 ⋅ x^2 + a_4 ⋅ x^4 + … + an-2 ⋅ xn-2) + x ⋅ (a_1 + a_3 ⋅ x^2 + a_5 ⋅ x^4 + … + an-1 ⋅ xn-2)

令

A1(x) = (a_0 + a_2 ⋅ x + a_4 ⋅ x^2 + … + an-2 ⋅ x \frac{n-2}{2})

A2(x) = (a_1 + a_3 ⋅ x + a_5 ⋅ x^2 + … + an-1 ⋅ x \frac{n-2}{2})

则可得到

A(x) = A1(x) + x ⋅ A2(x^2)

分类讨论

设 0 ≤ k ≤ \frac{n}{2} - 1 ， k ∈ Z

A(ωnk) = A1(ωn2k) + ωnk ⋅ A2(ωn2k)

由上文提到的折半引理

A(ωnk) = A1(ω\frac{n}{2}k) + ωnk ⋅ A2(ω\frac{n}{2}k)

对于 \frac{n}{2} ≤ k + \frac{n}{2} ≤ n - 1

A(ωnk + \frac{n}{2}) = A1(ωn2k + n) + ωnk + \frac{n}{2} ⋅ A2(ωn2k + n)

其中 ω_n2k + n = ωn2k ⋅ ωnn = ωn2k = ω\frac{n}{2}k

由消去引理 ωnk + \frac{n}{2} = - ωnk

故 A(ωnk + \frac{n}{2}) = A1(ω\frac{n}{2}k) - ωnk ⋅ A2{ω\frac{n}{2}k}

注意， k 与 k + \frac{n}{2} 取遍了 [0, n-1] 中的 n 个整数，保证了可以由这 n 个点值反推解出系数（上文已证明）。

于是我们可以知道

如果已知了 A1(x), A2(x) 分别在 ω\frac{n}{2}0, ω\frac{n}{2}1, …, ω\frac{n}{2}\frac{n}{2} - 1 的取值，可以在 O(n) 的时间内求出 A(x) 的取值

而 A1(x), A2(x) 都是 A(x) 一半的规模，显然可以转化为子问题递归求解。

考虑到分治 DFT 能处理的多项式长度只能是2^m(m ∈ N*)，否则在分治的时候左右不一样长，右边就取不到系数了。所以要在第一次 DFT 之前就把序列向上补成长度为2^m(m ∈ N*)(高次系数补0)、最高项次数为2^m - 1的多项式

在代入值的时候，因为要代入n个不同值，所以我们代入ω_n^0, ω_n^1, ω_n^2, … ω_nn-1 (n = 2^m(m ∈ N*))一共2^m个不同值

代码实现方面，STL 提供了复数的模板，当然也可以手动实现。两者区别在于，使用STL的complex可以调用exp函数求出ω_n。但事实上使用欧拉公式得到的虚数来求ω_n也是等价的

时间复杂度：T(n) = 2T(\frac{n}{2}) + O(n) = O(nlogn)

离散傅里叶反变换IDFT

使用快速傅里叶变换将点值表示的多项式转化为系数表示，这个过程叫做离散傅里叶反变换（Inverse Discrete Fourier Transform）

即由 n 维点值向量 (A(x_0), A(x_1), A(x_2), …, A(xn-1)) 推出 n 维系数向量(a_0, a_1, a_2, …, an-1)

设(d_0, d_1, d_2, …, dn-1) 为 (a_0, a_1, a_2, …, an-1)得到的离散傅里叶变换的结果。

我们构造一个多项式F(x) = d_0 + d_1 ⋅ x + d_2 ⋅ x^2 + … + dn-1 ⋅ xn-1

设向量(c_0, c_1, c_2, … cn-1) 中 c_k 为F(x)在x = ωx-k的点值表示

即c_k = ∑i=0n-1di ⋅ (ωn-k)^i

我们考虑对d_i还原，于是 c_k = ∑i=0n-1[ ∑j=0n-1a_j ⋅ (ωni)^j] ⋅ (ωn-k)^i

由和式的性质，c_k = ∑j=0n-1 a_j ∑i=0n-1(ω_n^i)^j ⋅ (ωn-k)^i = ∑j=0n-1 a_j ∑i=0n-1(ω_n^i)j-k

令S(j, k) = ∑i=0n-1(ωni)j-k

对其进行化简，设j - k = δ ，则S(j, k) = ωn0 + ωnδ + ωn2δ + … + ωn(n-1)δ

其公比为ωnδ，当ωnδ = 1 即 δ = 0 时，S(j, k) = n, 此时δ = 0 ⇒ j - k = 0 ⇒ j = k

当ωnδ ≠ 1 即 δ ≠ 0 时

由等比数列求和公式 S(j, k) = \frac{ωn0[(ωnδ)^n - 1]}{ωnδ - 1} = \frac{1[(ωnn)δ - 1]}{ωnδ - 1} = \frac{[(1)δ - 1]}{ωnδ - 1} = \frac{0}{ωnδ - 1} = 0，此时j ≠ k

所以S(j, k) = [j = k] ⋅ n

将S(j, k)带入原式

c_k = ∑j=0n-1a_j ⋅ S(j, k) = ∑j=0n-1a_j ⋅ [j = k] ⋅ n = a_k ⋅ n

所以a_k = \frac{c_k}{n}

其中a_k为原多项式A(x)的系数向量(a_0, a_1, a_2, …, a_n)中的a_k

由此得到：对于多项式A(x)由插值节点(ωn0, ωn1, ωn2, …, ωnn-1)做离散傅里叶变换得到的点值向量(d_0, d_1, d_2, …, dn-1), 我们将(ωn0, ωn-1, ωn-2, …, ωn-(n-1))作为插值节点，(d_0, d_1, d_2, …, dn-1)作为系数向量，做一次离散傅里叶变换得到的向量每一项都除以n之后得到的(\frac{c_0}{n}, \frac{c_1}{n}, …, \frac{cn-1}{n})就是多项式的系数向量(a_0, a_1, a_2, …, an-1)

注意到ωn-k是ωkn的共轭复数

\[

\begin{bmatrix}y[0] \ y[1] \ y[2] \ y[3] \ \dots \ y[n-1] \end{bmatrix} \begin{matrix}= \ = \ = \ = \ \ = \end{matrix} \begin{bmatrix}1 & 1 & 1 & 1 & \dots & 1 \ 1 & \omega_n^1 & \omega_n^2 & \omega_n^3 & \dots & \omega_n^{n-1} \ 1 & \omega_n^2 & \omega_n^4 & \omega_n^6 & \dots & \omega_n^{2(n-1)} \ 1 & \omega_n^3 & \omega_n^6 & \omega_n^9 & \dots & \omega_n^{3(n-1)} \ \dots & \dots & \dots & \dots & \dots & \dots \ 1 & \omega_n^{n-1} & \omega_n^{2(n-1)} & \omega_n^{3(n-1)} & \dots & \omega_n^{(n-1)^2} \end{bmatrix} \begin{bmatrix} a[0] \ a[1] \ a[2] \ a[3] \ \dots \ a[n-1] \end{bmatrix}

\]

这个过程称为离散傅里叶反变换
