时间戳

可以按照深度优先搜索的时间顺序，依次给予这 \(N\) 个点 \(0 \sim N - 1\) 的标记，该标记就被称为时间戳，记为 \(dfn\)。

树的DFS序

在刚进入递归后以及即将回溯前各记录一次该点的编号，最后产生的长度为 \(2N\) 的节点序列就称为树的 \(DFS\) 序。

\(DFS\) 序的特点是：每个节点的编号在序列中出现两次，设这两次出现的位置是 \(L[x]\) 与 \(R[x]\)，那么闭区间 \([L[x], R[x]]\) 就是以 \(x\) 为根的子树的 \(DFS\) 序，这使我们在很多与树相关的问题中，可以通过 \(DFS\) 序把子树统计转化为序列上的区间统计。

         0
        /|\
       1 6 3
      /|  / \
     7 4 2   5
          \
           8

DFS序：0, 1, 7, 7, 4, 4, 1, 6, 6, 3, 2, 8, 8, 2, 5, 5, 3, 0
          |--------------|  |--|  |--------------------|   
                子树1       子树6         子树3

树的深度

树中各个节点的深度是一种自顶向下的统计信息，起初，已知根结点的深度为 \(0\)，若节点 \(x\) 的深度为 \(d[x]\)，则它的子节点 \(y\) 的深度就是 \(d[y] = d[x] + 1\)。

图的连通块

进行深度优先搜索可以将图的连通块划分，也可以划分出森林中的每棵树。
