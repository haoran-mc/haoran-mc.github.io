wordcloud库函数

函数 作用
font_path:string 字体路径，例如font_path = '黑体.ttf'
width:int(default=400) 输出的画布宽度，默认为400像素
height:int(default=200) 输出的画布高度，默认为200像素
mask:nd-array or None (default=None) 参数为空：使用二维遮罩绘制词云；非空：设置的宽高值忽略，遮罩形状被mask取代
min_font_size:int(default=4) 显示的最小的字体大小
max_words:number(default=200) 要显示的词的最大个数
stopwords:set of strings or None 设置需要屏蔽的词，如果为空，使用内置的STOPWORDS
background_color:color value (default='black') 设置背景颜色
max_font_size:int or None(default=None) 显示的最大的字体大小
relative_scaling:float(default=.5) 词频和字体大小的关联性
color_func:callable, defaule=None 生成新颜色的函数，如果为空，则使用self.color_func
fit_words(frequencies) 根据词频生成词云(frequencies为字典类型)
generate(text) 根据文本生成词云
generate_from_frequencies(frequencies[, …]) 根据词频生成词云
generate_from_text(text) 根据文本生成词云
process_text(text) 将长文本分词并去处屏蔽词(如果是中文，需要先使用别的库实现
to_array() 转化为numpy array
to_file(filename) 输出到文件
