操作DOM对象

DOM: 文档对象模型(文档树)

浏览器网页就是一个 DOM 树形结构！

  • 更新: 更新DOM节点
  • 遍历DOM节点: 得到DOM节点
  • 删除: 删除一个DOM节点
  • 添加: 添加一个新节点

要操作一个 DOM 节点，就要先获得 DOM 节点

获得节点

var eleh1 = document.getElementsByTagName('h1');
var elep1 = document.getElementById('p1');
var elep2 = document.getElementsByClassName('p2');
var h = 123;
var father = document.getElementById('father');

更新节点

var id1 = document.getElementById('id1');
id1.innerText='123';   // 插入文本
id1.innerHTML='<strong>123</strong>'   // 插入超文本
id1.style.color = 'red';  // 设置CSS
id1.style.fontSize = '200px';
id1.style.padding = '2em';

删除节点

先获取父结点，再通过父结点删除子结点

var self = document.getElementById('p1');
var father = p1.parentElement;
father.removeChild(p1);
father.removeChild(father.children[0]);
father.removeChild(father.children[1]);  // 这个时候删除的实际上是开始时的第三个节点

插入节点

获得某个 Dom 节点后，如果这个节点是空的，可以通过 innerHTML 增加一个元素；如果这个 Dom 节点已经存在元素了，再使用 innerHTML 就会覆盖原来的元素。

追加节点，

    <p id="js">JavaScript</p>
    <div id="list">
    <p id="se">JavaSE</p>
    <p id="ee">JavaEE</p>
    <p id="me">JavaME</p>
    </div>

// 增加节点
var js = document.getElementById('js');
var list = document.getElementById('list');
list.apprndChild(js);

var newP = document.createElement('p'); // 创建一个p标签
newP.id = 'newP';
newP.innerText = 'hello';
list.appendChild(newP);

var myScript = document.createElement('script');
myScript.setAttribute('type', 'text/javascript');
list.appendChild(myScript);

// 创建一个style标签
var myStyle = document.createElement('style');
myStyle.setAttribute('type', 'text/css');
myStyle.innerHTML = 'body{background-color: red;}';
document.getElementByTagName('head')[0].appendChild(myStyle);
