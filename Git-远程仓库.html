远程仓库

  1. 新创建一个远程仓库
  2. SSH密钥
  3. git remote add origin git@github.com:haoran-mc/Practice_git.git
  4. git push -u origin master
  5. 删除远程库：
    git remote -v
    查看远程信息
    git remote rm <name>
    git remote rm origin

可以关联 github 的同时关联 gitee git remote add github git@github.com:haoran-mc/Practice_git.git git remote add gitee git@gitee.com:haoran_mc/Practice_git.git

git remote -v 查看远程库信息就能看到两个远程库

提交的时候要表明要提交的远程库和分支 git push github master git push gitee master
