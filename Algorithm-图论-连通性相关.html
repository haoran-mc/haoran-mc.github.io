有向图的连通性

强连通
在有向图中，如果两个点u、v是互相可达的，即从u出发可以到达v，从v出发可以到达u，则称u和v是强连通的，如果图中任意两个点都是互相可达的，称这个图是强连通图
强连通分量
如果一个有向图不是强连通图，那么可以把它分成多个子图，其中每个子图的内部是强连通的，而且这些子图已经扩展到最大，不能与子图外的任意点强连通，称这样的一个"极大强连通"子图是图的一个强连通分量(Strongly Connected Conmonent)

求SCC的3种高效算法

Kosaraju

利用了反图的技术

Tarjan

定理: 一个SCC，从其中任何一个点出发，都至少一条路径能绕回自己
这个定理结合求割点的num[]和low[]操作完成Tarjan算法

双连通分量

点双连通分量

在一个连通图中选任意两点，如果他们之间至少存在两条"点不重复"的路径，称为点双连通，一个图中的点双连通极大子图称为"点双连通分量"，点双连通分量是一个可靠的图，去掉任意一个点，其他点仍然是连通的，也就是说，点双连通分量中没有割点
求解点双连通分量和求割点密切相关，不同的点双连通分量最多只有一个个公共点，即某个割点，任意一个割点都是至少两个点双连通分量的公共点
求解点双连通分量一般用到Tarjan算法(Tarjan提出的算法之一)
在使用DFS计算割点时，找到一个割点时就已经完成了一次对某个极大点双连通子图的访问，那么在进行DFS的过程中，把遍历过的点保存起来，就可以得到这个点双连通分量

边双连通分量

边双连通分量的计算用到了"缩点"的技术

  1. 首先找出图中所有的边双连通分量
    在DFS过程中，图中所有的点都生成low值，low值相同的点必定在同一个边双连通分量中，DFS结束后，有多少low值就有多少个边双连通分量
  2. 把每个边双连通分量都看做一个点，即把那些low值相同的点合并成为一个缩点，这些缩点形成了一棵树
  3. 问题转化为: 至少在缩点树上增加多少条边才能使这棵树变为一个边双连通图，容易推导出: 至少增加的边数 = (总度数为1的结点数 + 1) / 2

割点和桥

割点
在一个连通分量中有一些关键的点，如果删除它，会把这个连通分量分成两个或更多，这种点称为割点
割边
在一个连通分量中，如果删除一个边，把这个连通分量分成了两个，这个边称为割边

使用DFS求割点，即利用"深搜优先生成树"求割点
在一个连通分量G中，对任意一个点s做DFS，能访问到所有的点，产生一棵"深搜优先生成树"T

a --- b                      a
| \                         /|\
|  \                       c | b
|   \                     /  |
c     d                  e   |
|     |                 / \  |
|     |                g   f |
e --- f                     \|
|                            d
|
g

图中c点，它的子结点只有一个e，而e后面有一个子结点d有回退边连回了根结点a，所以c不是割点
而e点，有一个子结点g，没有回退边连回e的祖先，所以e是割点
