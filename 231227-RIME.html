什么是？

什么是输入法引擎？

输入法引擎是实现中文输入法核心功能的软件组件，负责处理用户输入的拼音、笔画或手写，并将其转换为汉字或词语。

例子：搜狗输入法引擎、百度输入法引擎。

输入法引擎是非常关键的，它决定了你的输入体验。

什么是输入法？

输入法是用户使用的具体工具，它提供了用户界面，允许用户通过拼音、手写或其他方式输入中文字符。输入法通常包含与用户交互的界面元素，如软键盘或手写板。

例子：搜狗输入法、百度输入法。

输入法可以理解为 GUI 界面。

什么是输入法框架？

输入法框架是提供输入法和应用程序之间交互的软件架构。它定义了输入法与系统及应用程序之间的接口，确保输入法能够正确地接入系统，与各种应用程序协同工作。在Android中，输入法框架是一个系统级服务，而在Windows中，输入法框架是操作系统提供的组件。

例子：

  • Android 输入法框架
    • Google拼音输入法
    • 搜狗输入法
  • Windows 输入法框架
    • 微软拼音输入法
    • 搜狗输入法
  • macOS 输入法框架
    • 搜狗输入法
    • 百度输入法
  • Linux 输入法框架
    • fcitx输入法框架
    • ibus输入法框架
  • iOS 输入法框架
    • 谷歌拼音输入法
    • 搜狗输入法

什么是码表？

字典，实现编码到文字转换的字典数据。

例子：nihao 会转化为『你好』

安装 RIME

Mac

1. 安装「鼠须管」输入法

brew install --cask squirrel

2. 下载输入法

添加输入法方案需要用到输入法方案安装工具 plum。

# 克隆位置随意
git clone https://github.com/rime/plum.git

cd plum
# 需要代理
bash rime-install --select :all lotem/rime-forge/lotem-packages.conf

然后它会列出所有可以安装的方案，让你选择要安装哪些。

首先 prelude 和 essay-八股文是一定要安装的，因为是其他方案的基础，它们不是输入法方案，而只是配置文件。

然后我会再安装小鹤双拼输入法 double-pinyin（里面包含多种双拼输入法）。

Installing: double_pinyin.schema.yaml       # 自然码双拼
Installing: double_pinyin_abc.schema.yaml   # abc双拼
Installing: double_pinyin_flypy.schema.yaml # 小鹤双拼
Installing: double_pinyin_mspy.schema.yaml  # 微软双拼
Installing: double_pinyin_pyjj.schema.yaml  # 拼音加加双拼

3. 添加输入法

添加输入方案需要修改 default.yaml，但是我们不直接修改 default.yaml 文件，而是新建一个 default.custom.yaml 文件。

# ~/Library/Rime/default.custom.yaml

patch:
  # 方案列表（即输入法列表）
  schema_list:
    - schema: double_pinyin_flypy # 小鹤双拼

      # 设置按左shift键上屏并切换到英文输入状态
      ascii_composer/switch_key/Shift_L: commit_code
      # 设置按右shift键上屏并切换到英文输入状态
      ascii_composer/switch_key/Shift_R: commit_code

4. 修改简体

在 ~/Library/Rime 目录下新建 double_pinyin_flypy.custom.yaml 文件。

patch:
  switches:
    - name: ascii_mode
      reset: 0
      states: [ 中文, 西文 ]
    - name: full_shape
      states: [ 半角, 全角 ]
    - name: simplification
      reset: 1
      states: [ 漢字, 汉字 ]
    - name: ascii_punct
      states: [ 。，, ．， ]

其实就是在 [ 漢字, 汉字 ] 前面增加一行 reset: 1 。
