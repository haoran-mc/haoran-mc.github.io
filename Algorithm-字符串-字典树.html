字典树定义

             _
            |_|  root
            /|\
           / | \
          /  |  \
     [t] t   a   i [i]
        / \       \ 
       /   \       \
      /     \       \
[to] o       e [te]  n [in]
            /|\       \
           / | \       \
          /  |  \       \
         a   d   n       n [inn]
     [tea] [ted] [ten]

每个单词的结束位置对应一个"单词结点"，从根结点到每个单词结点的路径上所有字母连接而成的字符串就是该结点对应的字符串 ch[i][j]保存结点i的那个编号为j的子结点，ch[i][0] == 0表示结点i的子结点a不存在

  3. 字符串排序
  4. 前缀匹配
