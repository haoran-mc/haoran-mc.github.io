PYTHON-字典

  1. 字典的创建
    • 直接创建字典，将若干组键值对放在{}中创建字典，{键1：值1，键2：值2，…}
    • 使用内置函数dict()，将一组双元素序列转换为字典
      • list = [('键1', 值1), ('键2', 值2), ('键3', 值3), …]
      • dic = dict(list)
      • 存储双元素的可以使元组也可以是列表，但是一定只能包含两个元素，虽然字典允许值有多个(用元组或列表包含)，或者键有多个(只能用用元组包含)
    • 键具有唯一性，字典中不允许出现相同的键
    • 字典中的键必须是不可变的类型，所以键只能用元组包含多个元素，不能使用可修改的列表
  2. 访问字典
    • 字典是无序的，字典不能通过索引访问条目，访问字典只能通过键访问
    • 但字典中的值是一个序列，可以通过索引值访问
    • dic['键1']\[1]
  3. 字典更新
    • 添加条目，字典名[键] = 值
    • 修改条目，字典名[键] = 值
  4. 删除字典条目
    • del 字典名[键]
    • 字典名.pop(键, 默认值)
      • dic.pop(键1, "找不到要删除的条目")
      • 如果找到键1，会和列表中的pop一样返回键1对应的值，如果找不到，返回"找不到要删除的条目"
      • 与列表中不同的是，列表中默认pop()括号中是-1，而字典里没有索引的概念，如果缺省参数键，会报错
    • 字典名.popitem()
      • 无参，删除最后一个添加进字典的条目，删除的条目会以元组的形式返回
    • 字典名.clear()，清空所有条目变成一个空字典
    • del 字典名，删除整个字典
  5. 查找字典条目
    • 键 in 字典，存在返回True，不存在返回False
    • 字典名.get(键, 默认值)

  6. 字典的遍历 

    # 1. keys()
for key in dic.keys():
print(key, dic[key])

# 2. values()
for value in dic.values():
print(value)

# 3. items()
for item int dic.items():
print(item)
  7. 字典的排序
    • 使用sorted()函数，sort()函数无法使用
    • 返回语序的列表 list = sorted(dic)
      • 如果要对值进行排序?

        • 键值对交换 

          dic = {'Russia': 1707.5, 'Canada': 997.1, 'China': 960.1}
lis = [(value, key) for key, value in dic.items()]
lis.sort()
lis = [(key, value) for value, key in dic.items()]
print(lis)

  8. 字典的合并 

    # 1. 使用for循环
for key, value in dic1.items():
dic2[key] = value

# 2. 字典名.update(参数字典名)
dic2.update(dic1)

# 3. dict()
list1 = list(dic1.items()) + list(dic2.items())
dic1  = dict(list1)

# 4. dict()
dic1 = dict(dic1, **dic2)
