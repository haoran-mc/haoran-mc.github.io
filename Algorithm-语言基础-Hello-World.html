编译器

Windows

推荐使用GNU编译器，需要去 MinGW Distro 下载 MinGW 并安装。

Linux

使用 g++ -v 来检查是否安装过 g++

使用如下命令安装：

sudo apt-get install g++   # ubuntu
sudo yum install g++       # centos
sudo pacman -S g++         # manjaro

在命令行中编译代码

g++ test.cpp -o test && ./test

第一份代码

C++ 语言

#include <iostream>
using namespace std;
int main() {
    cout << "Hello World!" << endl;
    return 0;
}

C 语言

#include <stdio.h>
int main() {
    printf("Hello World!\n");
    return 0;
}
