新建用户

sudo su root 进入超管权限

useradd acm 创建用户acm
userdel -r acm 删除用户acm
passwd acm 给用户设置密码

cat /etc/passwd 查看用户
cat /etc/group 查看用户组
w 查看当前活跃用户

su acm 切换用户为acm
su root 进入root用户

给普通用户赋予root权限

  1. sudo vim /etc/sudoers
    root ALL=(ALL) ALL
    acm ALL=(ALL) ALL
  2. usermod -g root acm
