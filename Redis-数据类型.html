基本数据类型

String

append key01 value01
向 key01 后扩充字符串 "value01"
set key01 value01
append key01 value01
get key01   # value01value01
strlen key01
获取 key01 的长度
incr num01
对 num01 自增，num01 必须是数字才可以，否则 error
set num01 0
incr num01
incr num01
get num01   # "2"
decr num01
对 num01 自减
incrby num01 10
num01 += 10
decrby num01 10
num01 -= 10
getrange key01 0 3
截取 key01 的值从 0 到 3
set key01 value01
getrange key01 0 3    # valu
getrange key01 0 -1   # 查看全部的字符串
setrange key01 2 --
从 key01 的第 3 个字符开始替换
set key01 value01
get key01   # value01
setrange key01 2 --
get key01   # va--e01
setex key01 30 value01
set with expire 同时设置过期时间
setnx key02 value02
set if not exist 不存在时才设置键值对
setex key01 30 value01
ttl key01   # 27

setnx key02 value03   # (integer) 0
get key02             # "value02"
mset k1 v1 k2 v2 k3 v3

mget k1 k2 k3
批量获取值

get k4               # (nil)
getset key01 value02
先获取再设置

Mylist

使用 Mylist 可以完成栈、队列等的操作。

lpush mylist one
向 mylist 中插入 "one"
lrange mylist 0 -1
查看 mylist 中的所有值
lpush mylist one
lpush mylist two
lpush mylist three
lrange mylist 0 -1
# three two one
rpush mylist right
向 mylist 中插入 "right"，但是插入的方向与 lpush 不同
lpop mylist
弹出 mylist 的第一个元素
rpop mylist
弹出 mylist 的最后一个元素
lindex mylist 1
通过下标获取值
llen mylist
查看 mylist 的长度
lrem mylist 1 one
移除 mylist 中的一个 one
lrem mylist 2 three
移除 mylist 中的两个 three
ltrim mylist 1 2
截断，只保留 [1, 2] 这个下标区间的值
rpoplpush mylist othermylist
移除 mylist 列表的最后一个元素并放在 othermylist 第一个位置
lset mylist 0 item
将 mylist 的第 0 个替换为 "item"（redis 会自动识别字符串，所以加不加双引号无所谓），mylist 不能为空
linsert mylist before "world" "hello"
在 mylist 中 "world" 单词的前面插入 "hello"
linsert mylist after "hello" "!"
在 mylist 中 "hello" 单词的后面插入 "!"
lpush mylist world
linsert mylist before "world" "hello"
lrange mylist 0 -1

Set

set 中的值是不能重复的。

sadd myset "hello"
向 myset 中添加一个元素 "hello"
smembers myset
查看 myset 中的所有值
sismember myset hello
判断 myset 中是否有 "hello"
scard myset
查看 myset 中的元素个数
srem myset hello
移除 myset 中的 "hello"
srandmember myset
随机得到 myset 中的一个元素
srandmember myset 2
随机得到两个
spop myset
随机弹出 myset 中的一个元素
smove myset otherset "hello"
把 myset 中的 "hello" 移动到 otherset 中
sdiff set01 set02
set01 与 set02 的差集
sadd set01 a
sadd set01 b
sadd set01 c
sadd set02 c
sadd set02 d
sadd set02 e

sdiff set01 set02
# "b" "a"
sinter set01 set02
交集
sinter set01 set02
# "c"
sunion set01 set02
并集
sunion set01 set02
# "a" "b" "c" "d" "e"

hget myhash name
获取 name 对应的的值
hmset myhash field01 value01 field02 value02 field03 value03
多组设置
hmget myhash field01 field02 field03
多组获取
hgetall myhash
获取全部的键值对
hdel myhash field01
删除指定键值对
hlen myset
查看长度
hexists myhash field01
判断是否存在
hkeys myhash
获取所有的 key
hvals myhash
获取所有的 value
hincrby myhash field03 1
给 field03 的值自增 1
hdecrby myhash field03 1
给 field03 的值自减 1
hsetnx myhash field04 hello
不存在 field04 时才创建

Zset

有序集合，在 set 的基础上增加了一个值（collection score value），用于排序。

zadd myset 1 one
添加一个值
zadd myset 2 two 3 three
添加多个值
zrange myset 0 -1
查看所有数据
zadd salary 1500 peo01
zadd salary 500  peo02
zadd salary 1000 peo03
zrangebyscore salary -inf +inf
按 score 排序
zrangebyscore salary -inf +inf withscores
按 score 排序，同时输出 score
zreverange salary 0 -1
从大到小排序
zrem salary peo03
移除 peo03
zcard salary
查看长度
zcount myset 1 3
1 ~ 3 之间有几个值

特殊数据类型

geospatial

# geoadd 添加地理位置
# 两级无法直接添加
# key value（纬度、经度、名称）
# 有效的经度从 -180 到 180
# 有效的纬度从 -85.05112878 到 85.05112878
geoadd china:city 116.40 39.90 beijing
geoadd china:city 121.47 31.23 shanghai
geoadd china:city 106.50 29.53 chongqing
geoadd china:city 114.05 22.52 shenzhen
geoadd china:city 120.16 30.24 hangzhou
geoadd china:city 108.96 34.26 xian

# geopos 获取定位
geopos china:city beijing
geopos china:city beijing chongqing

# geodist 返回两个位置之间的距离
# 单位：
# - m 米
# - km 千米
# - mi 英里
# - ft 英尺
geodist china:city beijing shanghai
geodist china:city beijing shanghai km

# georadius 根据给定的经纬度为中心，找出某一半径内的元素
georadius china:city 110 30 500 km
# chongqing
# xian
georadius china:city 110 30 500 km withdist
# chongqing 341.9374
# xian 483.8340
georadius china:city 110 30 500 km withcoord
# chongqing 106.499 29.52
# xian 108.96 34.25
georadius china:city 110 30 500 km withdist whithcoord count 1
# 限定只查一个
# chongqing 341.9374 106.499 29.52

georadiusbymember china:city beijing 1000 km
# beijing
# xian
georadiusbymember china:city shanghai 400 km
# hangzhou
# shanghai

# geohash 返回一个或多个位置元素的 geohash 表示
# 11 个字符的 geohash 字符串
geohash china:city beijing
# wx4fbxxfke0
geohash china:city beijing chongqing
# wm5xzrybty0

GEO 底层的实现原理是 Zset，可以使用 Zset 命令来操作 geo。

zrange china:city 0 -1
zrem china:city beijing

hyperloglog

什么是基数？

在数学上，是集合论中刻画任意集合大小的概念。

可以通俗的理解为不重复的元素个数。

hyperloglog 基数统计的算法！

官网给出，使用 hyperloglog 计数可能会有 0.81% 的错误率，但是对于 \(2^{64}\) 的数量的 uid，只需要 12kb 的内存。

所以如果允许容错，在占用内存极少的优秀特性下 hyperloglog 计数是可以接受的。

如果不允许容错，那就使用 set 或自己的数据结构。

pfadd mykey01 a b c d
# 添加值

pfcount mykey01
# 计算基数数量
# 4

pfadd mykey02 c d e f

pfmerge mykey03 mykey01 mykey02
# OK

pfcount mykey03
# 6

bitmaps

位存储。

setbit sign 0 1
# 设置 sign 的第 0 位为 1

setbit sign 1 0
# 设置 sign 的第 1 位为 0

setbit sign 2 0
# 设置 sign 的第 2 位为 0

# 可以用来记录周一到周日的打卡情况，是否打卡。

getbit sign 0
# 查看 sign 的第 0 位

getbit sign 1

bitcount sign
