康托展开

康托展开是一种特殊的哈希函数

状态 \(012345678\) \(012345687\) \(012345768\) \(012345786\) \(\dots\) \(876543210\)
\(Cantor\) \(0\) \(1\) \(2\) \(3\) \(\dots\) \(362880-1\)

函数 \(Cantor()\) 实现的功能是：输入一个序列，计算出他的Cantor值，时间复杂度为 \(O(n^2)\)。

康托展开原理：
判断 \(2143\) 是 \({1, 2, 3, 4}\) 的全排列中第几大的数。

  1. 首位小于 \(2\) 的所有排列，比 \(2\) 小的数只有 \(1\)，后面 \(3\) 个数的排列有 \(3!\) 个，\(1 \times 3! = 6\)。
  2. 首位为 \(2\)，第二位小于 \(1\) 的所有排列，无，\(0 \times 2! = 0\)。
  3. 前两位为 \(21\), 第 \(3\) 位小于 \(4\) 的所有排列，只有 \(3\) 一个 \((2134)\)，\(1 \times 1! = 1\)。
  4. 前三位为 \(214\)，第 \(4\) 位小于 \(3\) 的所有排列，无，\(0 \times 0! = 0\)。

康托展开公式（第X个序列的计算公式）：\(X = a[n] \times (n-1)! + a[n-1] \times (n-2)! + \dots + a[i] \times (i-1)! + \dots + a[2] \times 1! + a[1] \times 0!\)

\(a[i]\) 表示第 \(i\) 个位置上的数在未出现的元素中排在第几（注意是否从0开始）。
