介绍

康托展开是一种特殊的哈希函数，它可以求出一个序列的排名。

给出四个数 \(\{1, 2, 3, 4\}\)，在这四个数的全排列中，排名第一位的序列是 \(\{1, 2, 3, 4\}\)，排名最后一位的序列是 \(\{4, 3, 2, 1\}\)。

状态 \(012345678\) \(012345687\) \(012345768\) \(012345786\) \(\dots\) \(876543210\)
\(Cantor\) \(1\) \(2\) \(3\) \(4\) \(\dots\) \(A_9^9 = 362880\)

那么对于任意给出的一个排列，怎么求得它的排名呢？就可以使用康托展开。

实现过程

函数 \(Cantor()\) 实现的功能是：输入一个序列，计算出他的Cantor值，时间复杂度为 \(O(n^2)\)。

康托展开过程：（判断 \(\{2, 1, 4, 3\}\) 是 \(\{1, 2, 3, 4\}\) 的全排列中排名第几的排列）

  1. 计算首位小于 \(2\) 的所有排列数：比 \(2\) 小的数只有 \(1\)，后面 \(3\) 个数的排列有 \(3!\) 个，\(1 \times 3! = 6\) 个。
  2. 首位为 \(2\)，第二位小于 \(1\) 的所有排列数：无，\(0 \times 2! = 0\) 个。
  3. 前两位为 \(21\), 第 \(3\) 位小于 \(4\) 的所有排列数：只有 \(3\) 一个 \((2134)\)，\(1 \times 1! = 1\) 个。
  4. 前三位为 \(214\)，第 \(4\) 位小于 \(3\) 的所有排列数：无，\(0 \times 0! = 0\) 个。

康托展开公式（第 \(ranking\) 个序列的计算公式）：\(ranking = a[n] \times (n-1)! + a[n-1] \times (n-2)! + \dots + a[i] \times (i-1)! + \dots + a[2] \times 1! + a[1] \times 0!\)

\(a[i]\) 表示第 \(i\) 个位置上的数在未出现的元素中排在第几，也就是第 \(i\) 个位置后面位置上小于第 \(i\) 位置上数的个数 \(+ 1\)。

给出代码：

int Contor(const vector<int>& permutation) {
    int num = 0;
    int len = permutation.size();
    for (int i = 0; i < len; ++ i) {
        int cnt = 0;
        for (int j = i + 1; j < len; ++ j)
            if (permutation[i] > permutation[j])
                cnt ++ ;
        num += cnt * fact[len - i - 1];   // 阶乘
    }
    return num + 1;
}

逆康托展开

康托展开是双射的，所以给出一个排名，可以得到这个排名对应的序列。

给出代码：

vector<int> revContor(int ranking, int n) {
    ranking = ranking - 1;   // 有 ranking - 1 个排列比目标序列要小
    vector<bool> vis(n + 1, false);
    vector<int> permutation(n, -1);

    for (int i = 0; i < n; ++ i) {
        int r = ranking / (fact[n - i - 1]);
        ranking = ranking % (fact[n - i - 1]);

        for (int j = 1; j <= n; ++ j)
            if (!vis[j] && !(r -- )) {
                vis[j] = true;
                permutation[i] = j;
                break;
            }
    }
    return permutation;
}
