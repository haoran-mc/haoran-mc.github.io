1、jQuery 是什么？

jQuery 是一个以 Javascript 来编写的函数库，简单来说就是提供你很多 Javascript 的函数功能，用途是让开发者可以更轻松方便的制作网站功能，最重要的是它是免费的。

2、有可能使用 Javascript 做出和 jQuery 一样的功能吗？

当然可以，jQuery 就使用 Javascript 写成的，本质上还是 Javascript，只是需要花些时间而已。

3、为什么使用 jQuery？

  1. 轻量级，压缩包只有 85kb 左右；
  2. 强大的选择器，jQuery 不仅支持 CSS 里的所有选择器，还有 jQuery 独有的选择器；
  3. 出色的 DOM 操作封装；
  4. 兼容性好，jQuery 能够在不同的浏览器中兼容，同时还修复了一些浏览器之间的差异；
  5. 链式编程，对发生在同一个 jQuery 对象上的一组动作，可以直接连写而无需重复获取对象，使得 jQuery 的代码优雅；
  6. 行为与结构分离
  7. 丰富的插件支持
  8. 开源

4、下载使用 jQuery

https://jquery.com/
jQuery 官网，在官网中下载 js 文件，放在项目目录下就可直接引用使用。
<script type="text/javascript" src="./js/jquery-3.3.1.min.js"></script>

5、选择 jQuery 版本

  • 为了兼容性选择 1.x
  • 否则选择 3.x
