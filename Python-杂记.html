pip

pip 是 Python 包管理工具，该工具提供了对Python 包的查找、下载、安装、卸载的功能

  • pip3 search tkinter 查找tkinter这个包
  • pip3 list 将安装的包列出
  • sudo pip3 install wordcloud
  • sudo pip3 install jieba
  • pip3 install numpy
    • sudo apt-get install build-essential sudo apt-get install cmake git libgtk2.0-dev pkg-config libavcodec-dev libavformat-dev libswscale-dev
  • sudo pip3 install pandas
  • sudo apt update
    • sudo apt install python3-opencv
    • 验证安装
      • python3 -c "\
      • import cv2
      • print(cv2.__version__)"
  • sudo apt install cmake
  • sudo apt install cmake-curses-gui

pip使用详解

pip安装包

pip install 安装包名

pip查看是否已安装

pip show –files 安装包名

pip检查哪些包需要更新

pip list –outdated

pip升级包

pip install –upgrade 要升级的包名

pip卸载包

pip uninstall 要卸载的包名

tkinter，tcl，tk

  • Tcl 是“工具控制语言(Tool Command Language)”的缩写，其面向对象为otcl语言
  • Tk 是 Tcl“图形工具箱”的扩展，它提供各种标准的 GUI 接口项，以利于迅速进行高级应用程序开发
  • tkinter
    • 是Python调用tcl程序的标准Python程序，可以通过这个interface调用tcl的程序，因为在大多数的unix系统中都内置了很多的tcl程序和命令
    • 是python自带的GUI库，是对图形库TK的封装
    • 是一个跨平台的GUI库，开发的程序可以在win，linux或者mac下运行
  • python3 -m tkinter //不知道这句话是干啥的，给出一个tkinter的例子？
  • sudo apt search python3-tk //查询是否安装python3-tk
  • sudo apt install python3-tk //如果没有安装就执行这条命令安装
  • sudo apt install tk-dev //安装tk的开发类库
