整数划分

一个正整数 \(n\) 可以表示成若干个正整数之和，形如：\(n = n_1 + n_2 + \dots + n_k\)，其中 \(n_1 \geq n_2 \geq \dots \geq n_k, k \geq 1\)。

我们将这样的一种表示称为正整数 \(n\) 的一种划分。

现在给定一个正整数 \(n\)，请你求出 \(n\) 共有多少种不同的划分方法。

例如给出一个正整数 \(5\)。

可以划分为：

5 = 5
5 = 4 + 1
5 = 3 + 2
5 = 3 + 1 + 1
5 = 2 + 2 + 1
5 = 2 + 1 + 1 + 1
5 = 1 + 1 + 1 + 1 + 1

完全背包

看成有一个体积为 \(n\) 的背包和体积分别为 \(1, 2, 3, \dots, n\) 的 \(n\) 种物品，物品的数量不限，问装满这个背包的选法有多少种。

状态表示：\(dp[i][j]\)：从 \(1 \sim i\) 的物品中恰好选出体积为 \(j\) 的选法的数量。

状态转移：\(dp[i][j] = dp[i - 1][j] + dp[i - 1][j - i] + dp[i - 1][j - 2i] + dp[i - 1][j - 3i] + \dots + dp[i - 1][j - si], (si \leq j)\)

即：

\(dp[i - 1][j]\)
从 \(1 \sim i - 1\) 的物品中恰好选出体积为 \(j\) 的选法的数量。
\(dp[i - 1][j - i]\)
从 \(1 \sim i - 1\) 的物品中恰好选出体积为 \(j - i\) 的选法的数量。
\(dp[i - 1][j - 2i]\)
从 \(1 \sim i - 1\) 的物品中恰好选出体积为 \(j - 2i\) 的选法的数量。
\(\dots\)
\(\dots\)
\(dp[i - 1][j - si]\)
从 \(1 \sim i - 1\) 的物品中恰好选出体积为 \(j - si\) 的选法的数量。

又有：\(dp[i][j - i] = dp[i - 1][j - i] + dp[i - 1][j - 2i] + dp[i - 1][j - 3i] + \dots + dp[i - 1][j - si]\)

所以：\(dp[i][j] = dp[i - 1][j] + dp[i][j - i]\)

计数DP

直接给出状态表示：\(dp[i][j]\)：所有总和是 \(i\)，并且恰好表示成 \(j\) 个数的和的方案数量。

分为两种情况：

  1. 这 \(j\) 个数中最小值是 \(1\)，那么把这个 \(1\) 给去掉，就有 \(j - 1\) 个数的总和是 \(i - 1\)。
  2. 这 \(j\) 个数中最小值不是 \(1\)，那么把这 \(j\) 个数都减 \(1\)，就还是 \(j\) 个数，总和变成了 \(i - j\)。

显然所有的数的组合都被这两种情况所涵盖，且不重合。

就有：\(dp[i][j] = dp[i - 1][j - 1] + dp[i - j][j];\)

最后的结果是 \(dp[n][1] + dp[n][2] + \dots + dp[n][n]\)。
