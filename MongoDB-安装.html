Linux 安装

# 在 manjaro 安装
yay -S mongodb-bin

下载后关于 mongodb 的文件分别所在的位置：

/usr/bin/mongo
mongodb 客户端脚本
/usr/bin/mongod
用来启动 mongodb 的启动脚本
/usr/bin/mongos
路由脚本
/etc/mongodb.conf
配置文件
/usr/lib/systemd/system/mongodb.service
服务文件

启动 MongoDB 服务

cd /usr/bin
进入 mongod 的目录
./mongod –help
查看帮助文档
./mongod –port=27017 –dbpath=/home/haoran/.mongo/data –logpath=/home/haoran/.mongo/logs/mongo.log –fork
端口 27017、数据目录、日志写入的文件、后台运行，其中两个目录都要保证存在
ps aux|grep mongo
查看进程

客户端连接

cd /usr/bin
进入 mongo 的目录
(no term)
./mongo –port=27017

服务命令

sudo systemctl enable mongodb
开机启动
sudo systemctl status mongodb
查看状态
sudo systemctl restart mongodb
重启
sudo systemctl stop mongodb
停止
sudo systemctl daemon-reload
重载服务

Docker 安装

docker pull mongo
拉取 mongodb 镜像
docker run -d –name mongo01 -p 27017:27017 mongo
后台运行 mongo 镜像
docker exec -it mongo01 bash
进入 mongo 容器
