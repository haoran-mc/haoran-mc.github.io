PYTHON-列表

  1. 字符串的定义方式：
    1. 单引号('')
    2. 双引号("")
    3. 三单引号(''' ''')，可以跨行
    4. 三双引号(""" """)，可以跨行
  2. 字符串的索引
    • 正向从左向右从0递增
    • 反向从右向左从-1递减
  3. 字符串的切片
    • [头下标，尾下标，步长]
    • 缺省头下标，从开始取子串
    • 缺省尾下标，取子串到结尾
    • 步长为正，正向取子串
    • 步长为负，反向取子串
    • 间隔为步长绝对值减一
  4. 内置的字符串运算符
    • +字符串拼接"hello" + "world" –> "helloworld"
    • *字符串复制"hello" * 3 –> "hellohellohello"
    • in判断是否为子串

  5. 内置的字符串处理函数

    函数 描述
    len(x) 返回字符串x的长度
    str(x) 将任意类型x转换为字符串类型
    chr(x) 返回Unicode编码为x的字符
    ord(x) 返回x的Unicode编码值
    hex(x) 将整数x转换为十六进制数，并返回其小写字符串形式
    oct(x) 将整数x转换为八禁止数，并返回其小写字符串形式
  6. 内置的字符串处理方法
    1. 字符串查找类方法
      • find() 查找一个字符串在另一个字符串制定范围(默认是整个字符串)中首次出现的位置，如果不存在返回-1
      • rfind() 查找一个字符串在另一个字符串制定范围(默认是整个字符串)中最后一次次出现的位置，如果不存在返回-1
      • find() 查找一个字符串在另一个字符串制定范围(默认是整个字符串)中首次出现的位置，如果不存在抛出异常
      • rfind() 查找一个字符串在另一个字符串制定范围(默认是整个字符串)中最后一次次出现的位置，如果不存在抛出异常
      • find函数
        • 语法：str.find(sub[, start[, end]])
        • 参数： - sub：要查找的子串
          • start：起始查找位置
          • end：结束查找位置
        • 当sub在str中指定范围[start,end]\(可以省略，默认是[0, len(str)])中第一次匹配成功时位置的索引，否则返回-1
      • index函数
        • 语法：str.index(sub[, start[, end]])
        • 参数： - sub：要查找的子串
          • start：起始查找位置
          • end：结束查找位置
        • 当sub在str中指定范围[start,end]\(可以省略，默认是[0, len(str)])中第一次匹配成功时位置的索引，否则抛出异常ValueError
      • count() 返回一个字符串在另一个字符串中出现的次数，如果不存在则返回0
    2. 字符串分隔类方法
      • split()
        • str1.split(', ')
        • str2.split(maxsplit=2)
      • rsplit()
      • partition()
        • str1.partition('fish')
      • rpartition()
    3. 字符串连接方法
      • join()
        • '-'.join(str1)
        • ':'.join(str2)
    4. 大小写字符转换方法
      • lower()
      • upper()
      • capitalize()，首字母变大写
      • title()，字符串中每个单词首字母变大写
      • swapcase()，大小写互换
    5. 字符串替换方法
      • replace()
        • str.replace("big", "small")
    6. 删除字符串两端，右端或左端连续空白字符和制定字符方法
      • strip()
        • str.strip()，删除两端空白字符
        • str.strip('')，删除两端指定字符""
      • rstrip()
      • lstrip()
    7. 判断字符串是否以指定字符开始或结束方法
      • startswith()
        • str.startswith("Python") –> true or false
      • endswith()
    8. 判断字符串类型方法
      • isupper()，判断字符串是否为全大写
      • islower()，判断字符串是否为全小写
      • isdigit()，判断字符串是否全为数字
      • isalpha()，判断字符串是否全为字母
      • isalnum()，判断字符串是否全为数字或字母
    9. 字符串排版方法
      • center()
        • str.center(30, "")，字符串居中对齐，输出宽度为30，不足的用""填充
      • ljust()
        • str.ljust(30, "")，字符串居左对齐，输出宽度为30，不足的用""填充
      • rjust()
        • str.rjust(30, "")，字符串居右对齐，输出宽度为30，不足的用""填充
      • zfill()
        • str.zfill(30)，输出宽度为30，在字符串左侧以字符"0"填充
