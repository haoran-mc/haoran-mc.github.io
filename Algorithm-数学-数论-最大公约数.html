最大公因数

最大公约数即为 Greatest Common Divisor，常缩写为 gcd。

性质:

  1. \(gcd(a, b) = gcd(b, a)\)
  2. \(gcd(a, b) = gcd(a - b, b) (a >= b)\)
  3. \(gcd(a, b) = gcd(a \% b, b)\)
  4. \(gcd(a, b, c) = gcd(gcd(a, b), c)\)
  5. \(gcd(ka, kb) = k \times gcd(a, b)\)

辗转相除法(欧几里德算法)

定理：\(gcd(a, b) = gcd(b, a \% b)\)

已知两个数 \(a, b\) 如何求两者的最大公约数？

证：
因为 \(gcd(a, b) = gcd(b, a)\)，所以不妨设 \(a > b\)

设 \(z\) 为 \(a, b\) 的最大公约数，那么有 \(gcd(a, b) = z\)，\(\begin{cases} a = zm\\ b = zn \end{cases}\)

设 \(\frac{a}{b} = p \dots r\)
\(\rightarrow a = bp + r\)
\(\rightarrow zm = znp + r\)
\(\rightarrow z(m - np) = r = a \% b\)

因为 \(zn = b\)
所以若要证 \(b\) 与 \(a % b\) 的最大公约数也是 \(z\)，那么 \(n\) 与 \(m - np\) 互质

假设 \(n\) 与 \(m - np\) 不互质 \(\begin{cases} m - np = km'\\ n = kn' \end{cases}\)

代入得 \(m - pkn' = km'\)
\(\rightarrow \begin{cases}m = k(m' + n'p)\\ n = kn' \end{cases}\)

代入\(\begin{cases}a = zm\\ b = zn \end{cases}\) 有 \(\begin{cases}a = zk(m' + n'p)\\ b = zkn'\end{cases}\)

由假设得到 \(a, b\) 的最大公约数是 \(zk\)，矛盾，所以 \(m - np\) 与 \(n\) 互质，所以 \(b\) 与 \(r\) 的最大公约数是 \(z\)

所以 \(gcd(a, b) = gcd(b, a \% b)\)

代码实现

// gcd(a, b) = gcd(b, a mod b)
int gcd(int a, int b) {
    return b ? gcd(b, a % b) : a;
}

最小公倍数

两个数 \(a, b\) 的最小公倍数是 \(\frac{a \times b}{gcd(a, b)}\)。
