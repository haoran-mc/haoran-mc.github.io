关于web

  • web是基于HTTp协议进行交互的应用网络
  • web就是通过使用浏览器/APP访问的资源

目前主要的两种web开发的模式

  1. 将完整的html的返回给前端
  2. 前端的框架自己定义了模板，后端返回json格式的数据，前端通过json文件渲染

什么是 gin 框架

Gin 框架初体验

package main

import (
    "github.com/gin-gonic/gin"
    "net/http"
)

func main() {
    r := gin.Default()
    r.GET("/hello", func(c *gin.Context) {
        c.JSON(http.StatusOK, gin.H{
            "name": "小王子",
            "age": 18,
            "msg": "hello",
        })
    })
    r.Run(":9080")
}
