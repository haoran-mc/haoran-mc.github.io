邻接矩阵

  • 令G[i][j]的两维分别表示图的顶点标号，即G[i][j]表示顶点i与顶点j之间有边
  • 或者可以在G[i][j]中放边权
......................
|......0  1  2  3  4 |   
|                    | 邻 
| 0    0  2  0  0  1 | 接 
| 1    2  0  2  0  2 | 矩
| 2    0  2  0  1  0 | 阵
| 3    0  0  1  0  1 | 
| 4    1  2  0  1  0 |
......................
  • 由于需要开一个二维数组，如果顶点数目太大，便可能会超过题目限制的内存，因此邻接矩阵只适用于顶点数目不太大的题目

邻接表

  • 图G(V, E)的顶点编号为0, 1, … N-1，每个顶点都有可能有若干条出边，将同一个顶点的所有出边放在一个列表里，N个顶点有N个列表，这N个列表被称为图G的 邻接表
  • 由于需要开一个二维数组，如果顶点数目太大，便可能会超过题目限制的内存，因此邻接矩阵只适用于顶点数目不太大的题目
  • 这时候就需要使用邻接表
......head                                     | ==                V0 ------------- V1
.......__                                      |                   |             / |
Adj[0]|__| -> [1]\[2] -> [4]\[1]               | 邻              1 |           /   |
Adj[1]|__| -> [0]\[2] -> [2]\[2] -> [4]\[2]    | 接                |     2   /     |  2
Adj[2]|__| -> [1]\[2] -> [3]\[1]               | 表               V4 -------       |
Adj[3]|__| -> [2]\[1] -> [4]\[1]               |                   |               |
Adj[4]|__| -> [0]\[1] -> [1]\[2] -> [3]\[1]    | ==              1 |       1       |
...............................................|                  V3 ------------- V2
  • 建立邻接表可以使用链表，也可以使用更简单的vector
  • vector<int> Adj[N]; or vector<Node> Adj[N];
// 如果是无向边
adj[1].push_back(3);
adj[3].push_back(1);
