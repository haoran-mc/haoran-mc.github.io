PYTHON-MATH

python语言中的数值计算标准函数库math提供了4个数学常数和44个函数
math库不支持复数类型，仅支持整数和浮点数云端
  1. 引用math库的两种方式：

    1. import math

      import math
print(math.pi)

    2. from math import pi

      from math import pi
print(pi)

  2. math库中的常数

    常数 描述
    pi 圆周率，值为3.141592652589793
    e 自然对数，值为2.718281828459045
    inf 正无穷大，负无穷大-inf
    nan 非浮点数标记，Not a Number
    1. math库中的部分数值函数
    函数 描述
    fabs(x) 返回x的绝对值，只适用于float和integer类型，abs也适用于复数
    fmod(x, y) 返回x除y的余数
    fsum([x, y, …] 浮点数精确求和
    gcd(x, y) 返回x和y的最大公约数，x和y为整数
    trunc(x) 返回x的整数部分
    modf(x) 返回x的小数和整数两部分
    ceil(x) 向上取整，返回不小于x的最小整数
    floor(x) 向下取整，返回不大于x的最大整数
    factorial(x) 返回x的阶乘，x为整数
    1. math库中部分幂对数函数与三角函数
    函数 描述
    pow(x, y) 返回x的y次幂
    exp(x) 返回e的x次幂
    sqrt(x) 返回x的平方根
    log(x[,base] 返回x的对数值，只输入x时，返回lnx
    log2(x) 返回x的2对数值
    log10(x) 返回x的10对数值
    degress(x) x为弧度制，返回x对应的角度值
    redians(x) x为角度值，返回x对应的弧度值
    hypot(x, y) fanhui(x, y)坐标到原点(0, 0)的距离
    sin(x) 返回x的正弦函数值，x为弧度值
    cos(x) 返回x的余弦函数值，x为弧度值
    tan(x) 返回x的正切函数值，x为弧度值
    asin(x) 返回x的反正弦函数值，x为弧度值
    acos(x) 返回x的反余弦函数值，x为弧度值
    atan(x) 返回x的反正切函数值，x为弧度值
