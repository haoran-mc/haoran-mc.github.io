字符串拼接

使用 + 号

str = str1 + str2

使用 Sprintf 拼接

str = fmt.Sprintf("%s %d %s", str1, i, str2)

使用 join 函数拼接

var str0 []string = []string{str1, str2}
str := strings.Join(str0, "-")

使用 buffer.WriteString 函数拼接

var bt bytes.Buffer
bt.WriteString(str1)
bt.WriteString(str2)
str := bt.String()   // 获得拼接后的字符串

使用 buffer.Builder 拼接

var build strings.Builder
builder.WriteString(str1)
builder.WriteString(str2)
str := build.String()
