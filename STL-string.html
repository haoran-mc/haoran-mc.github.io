string常用函数

string-operator+=

  • 可以在string类型的变量与ch字符之间进行operator运算

string-compare operator

size

  • 返回string的长度
  • 时间复杂度O(1)

length

  • 与size()基本相同

insert

时间复杂度为o(1)

  • insert(position, string)
    • 在position的位置插入字符串string

  • insert(it, it_begin, it_end)

    • it为待插入位置
    • it_begin是插入字符串的首迭代器
    • it_end是插入字符串的尾迭代器
    • str1.insert(str1.begin()+3, str2.begin()+1, str2.end())
    //进制转换
void change(ill sum, int base) {
	while (sum != 0) {
		const char x = sum % base + '0';
		//要求是const char&
		res.insert(idx++, &x);
		sum /= base;
	}
}

erase

  • erase(it)
  • erase(first, last)
  • erase(pos, length) 时间复杂度均为o(N)

clear

  • clear()

  • 时间复杂度为O(1) 

    #include <iostream>
#include <string>
using namespace std;

int main() {
	string str = "abcdefg";
	str.clear();
	cout << str << endl;
	return 0;
}

substr

  • substr(pos, len)
  • 返回从position开始，长度为len的子串
  • 时间复杂度为O(len)
  • cout << str.substr(0, 5) << endl;

string::npos

  • string::npos是一个常数，其本身的值为-1，但由于是unsigned_int类型，因此实际上也可以认为是unsigned_int类型的最大值
  • string::npos用以作为find函数失配时的返回值

find

  • str1.find(str2)
    • 当str2是str1的子串时，返回其在str1中第一次出现的位置
      • AAPAATAAAA
      • str.find('P') == 2
      • str.find('T') == 5
      • str.length() == 10
    • 如果不是str1的子串，返回string::npos

  • str1.find(str2, pos)

    • 从str1的position位开始匹配str2

    时间复杂度O(nm) n, m分别是两串的长度 

    #include <iostream>
#include <string>
#include <vector>
using namespace std;

int main() {
	string str1 = "ababababababab";
	string str2 = "a";
	int pos = 0;
	vector<int> vec;

	printf("%d %d\n", (int)string::npos, (int)str1.length());

	while (pos != (int)string::npos && pos < (int)str1.length()) {
		pos = str1.find(str2, pos+1);
		if (pos != (int)string::npos)
			vec.push_back(pos);
	}
	for (int i = 0; i < (int)vec.size(); i++)
		printf("%d\n", vec[i]);

	return 0;
}

replace

  • str1.replace(pos, len, str2)
    • 把str从pos位开始，长度为len的子串替换为str2

  • str1.replace(it_begin, it_end, str2)

    • 把str的迭代器[it_begin, it_end)范围内的子串替换为str2

    时间复杂度为O(str1.length())

注意！

string类型的变量同样可以与字符进行相同操作

#include <cstdio>
#include <iostream>
#include <string>
using namespace std;

int main {
	string str = "ABCD";
	str += "E";
	cout << str << endl;
	str += 'F';
	cout << str << endl;
	return 0;
}

string中find函数可以在string类型的变量中查找一个字符

#include <cstdio>
#include <iostream>
#include <string>
using namespace std;

int main {
	string str = "ABCD";
	cout << str.find('C') << endl;
	return 0;
}

空的string类型的变量可以直接operator

#include <cstdio>
#include <string>
#include <iostream>
using namespace std;

int main {
	string str;
	str += "ABCD";
	cout << str << endl;
	str.clear;
	cout << str << endl;
	str += 'A';
	cout << str << endl;
	return 0;
}
