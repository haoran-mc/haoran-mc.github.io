UNORDERED_SET

set 头文件：#include <set> unordered_set 头文件：#intlude <unordered_set>

set 内部由红黑树实现 unordered_set 内部由散列实现

需要对输入的元素排序，使用set 不需要对输入元素排序，仅对于去重，使用unordered_set

  • 调用unordered_set的find()会返回一个迭代器。这个迭代器指向和参数哈希值匹配的元素，如果没有匹配的元素，会返回这个容器的结束迭代器st.end()
#include <cstdio>
#include <set>
#include <unordered_set>
using namespace std;
int main() {
		unordered_set<int> unst;
		for (int i = 0; i < 10; i++) {
				int num;
				scanf("%d", &num);
				unst.insert(num);
		}
		//好像是输入的逆顺序，不能确定
		for (unordered_set<int>::iterator it = unst.begin(); it != unst.end(); it++)
				printf("%d ", *it);

		return 0;
}
