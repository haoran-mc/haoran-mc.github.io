链表

链表可以通过指针+结构体实现，但在算法竞赛中，通常使用数组模拟堆栈，队列也是优先用数组模拟，STL虽好，但不宜重度依赖 

struct Node {
		int val;
		Node *nxet;

  1. 效率，使用指针时，每次都要new一个新的结点，这个太慢了
  2. 如果是预先创建好结点，那就和数组模拟没有差别

    • 遍历链表
    • 链表的插入

    建立链表的算法为：

    1. 生成一个结点并输入数据，如果数据域非空，则使头指针指向该结点，指针域置空。
    2. 生成新结点并输入数据，然后将该结点插入链表。
    3. 重复步骤(2)直到结束输入。
    4. 将尾结点的指针域置空。

链式前向星

邻接表的改进方法，用邻接表非常节省空间，然而如果空间及其紧张，可以使用邻接表的改进方法-链式前向星它是空间效率最高的存储方法，因为它用静态数组模拟邻接表，没有任何浪费 

// 这种的邻接表和一般的邻接表不一样，这个是按输入的倒序存储的
// 如果从head[x]开始搜索，得到的是最后一次加入到链表x的边tot
// vert[tot]，edge[tot]存储的是与x相连的节点和权值，然后进行操作
// i = next[i]，此时i是x结点上一次添加的边，所以是倒序打印
//加入有向边(x, y)，权值为z
void add(int x, int y, int z) {
		vert[++tot] = y;
		edge[tot]   = z;
		next[tot]   = head[x];
		head[x] = tot;
}
//访问从x出发的所有边
for (int i = head[x]; i; i = next[i]) {
		int y = vert[i];
		int z = edge[i];
		printf("%d %d %d\n", x, y, z);
}

上面的代码片段用数组模拟链表的方式存储了一张带权有向图的邻接表结构对于无向图，使用"成对变换"的位运算性质，如果vert[i]是第i条边的终点，那么vert[ixor1]就是第i条边的起点
