list的定义

Lists are sequence containers that allow constant time insert and erase operations anywhere within the sequence, and iteration in both directions.

resize

  • 调用resize(n)将list的长度改为只容纳n个元素，超出的元素将被删除。如果n比list原来的长度长，那么默认超出的部分元素置为0。也可以用resize(n, m)的方式将超出的部分赋值为m

clear

front

back

  • 通过front()可以获得list容器中的头部元素，通过back()可以获得list容器的最后一个元素。注意：当list元素为空时，这时候调用front()和back()不会报错。因此在编写程序时，需要先调用empty()函数判断list是否为空，然后调用front()和back()函数

assign

  1. a.assign(n, val):将a中的所有元素替换成n个val元素 

    list<int> lst{1, 2, 3, 4, 5};
lst.assign(5, 10);

  2. lst1.assign(lst2.begin(), lst2.end()) 

    list<int> lst1{6, 7, 8, 9};
list<int> lst2{1, 2, 3, 4, 5};
lst2.assign(lst1.begin(), lst1.end());

    #include <cstdio>
#include <list>
using namespace std;

void output(list<int> &lst) {
  for (auto it = lst.begin(); it != lst.end(); ++it)
    printf("%d ", *it);
  printf("\n");
}

int main() {
  list<int> lst1{0, 0, 1, 1, 2, 2};
  list<int> lst2{0, 1, 2, 3, 4, 5};
  lst2.assign(lst1.begin(), lst1.end());
  output(lst1);
  output(lst2);

  lst1 = {0, 0, 1, 1};
  lst2 = {0, 1, 2, 3, 4, 5};
  lst2.assign(lst1.begin(), lst1.end());
  output(lst1);
  output(lst2);

  lst1 = {0, 0, 1, 1, 2, 2};
  lst2 = {0, 1, 2, 3};
  lst2.assign(lst1.begin(), lst1.end());
  output(lst1);
  output(lst2);
  return 0;
}

swap

交换两个链表。a.swap(b)和swap(a, b)，都可以完成a链表和b链表的交换。 

list<int> lst1{6, 7, 8, 9};
list<int> lst2{1, 2, 3, 4, 5};
swap(lst1, lst2);
