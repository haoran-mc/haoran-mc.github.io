加权平均值

加权平均值即将各数值乘以相应的权数，然后加总求和得到总体值，再除以总的单位数。加权平均值的大小不仅取决于总体中各单位的数值（变量值）的大小，而且取决于各数值出现的次数（频数），由于各数值出现的次数对其在平均数中的影响起着权衡轻重的作用，因此叫做权数。

笛卡尔坐标系

笛卡尔坐标系(Cartesian coordinates)就是直角坐标系和斜坐标系的统称互相垂直且有公共原点的两条数轴构成平面直角坐标系，而如果坐标系中两条坐标轴不垂直，那么这样的坐标系称为“斜坐标系”

超长方体

超立方体是数学中立方体的四维类似物，所谓的点动成线，线动成面，面动成体。在四维空间(非三维-时间概念)中，立方体的移动形成四维的超立方体，由无数个立方体所组成的，具有四维的观念

圆的面积

由于编程里使用弧度制计算圆的相关数据，所以不再展示角度制的一些公式默认符号 角度：$α$，半径：r，母线：L，周长：C，面积：S，体积：V，圆周率：PI

  1. 计算角度：acos(d / R) （直角三角形斜边：R，所求角度两邻边的另一边：d）
  2. 计算圆弧的弧长：\(\alpha * r\)
  3. 计算扇形面积：\(\alpha * r * r / 2\)
  4. 计算圆的周长：2 * PI * r
  5. 计算圆的面积：PI * r * r

平方数

  • 平方和公式：\(1^2 + 2^2 + 3^2 + \dots + n^2 = \frac{n \times (n + 1) \times (2n + 1)}{6}\)
  • 立方和公式：\(1^3 + 2^3 + 3^3 + \dots + n^3 = {\frac{n (n + 1)}{2}}^2 = (1 + 2 + 3 + \dots + n)^2\)
  • 在$( 1 - ∞ )$中， 若要满足 \(( 1^2 + 2^2 + \dots + n^2 )\) 这个数为平方数，只能（n = 1 || n = 24）

组合数

  • \(C_n^0 = C_n^n = 1\)
  • \(C_n^m = \frac{n!}{m!(n-m)!} = (II)_{n-m}^n/2\)
  • \(C_n^m = C_n^{n-m}\)
  • \(C_n^m = C_{n-1}^m + C_{n-1}^{m-1}\)
  • \(kC_n^k = nC_{n-1}^{k-1} (1 <= k <= n)\)
  • \(A_n^m = C_n^m m! = \frac{n!}{(n-m)!}\) <!– 计算杨辉三角 –> 计算杨辉三角一行(第n-1行)所有数的值]
    • \(C_n^r = \frac{n - r + 1}{r} C_n^{r - 1}\)

等比数列

  • 求和公式 : \(S_n = \frac{a_1 (1 - q^n)}{1 - q} (q \neq 1)\)

坐标点的变换

坐标系旋转

x' = x * cos(t) + y * sin(t); y' = -x * sin(t) + y * cos(t);

坐标点旋转

x' = x * cos(t) - y * sin(t); y' = x * sin(t) + y * cos(t);
