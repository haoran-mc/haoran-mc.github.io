介绍

形如 \(ax \equiv c\ (mod\ b)\) 的方程被称为 线性同余方程(Congruence Equation)。

使用扩展欧几里德算法求解线性同余方程

定理1：方程 \(ax + by = c\) 与方程 \(ax \equiv c\ (mod\ b)\) 是等价的，有整数解的充要条件是 \(gcd(a, b) | c\)

\(ax \equiv c\ (mod\ b)\)
\(\Rightarrow \exists y\ st.\ ax = c + by\)
\(\Rightarrow ax - by = c\)
\(\Rightarrow ax + by' = c\)

求解线性同余方程转化为求解二元一次方程，使用扩展欧几里德算法求解，方程有解的充要条件是 \(gcd(a, b) | c\)

但是扩展欧几里德算法求得是 \(ax + by' = gcd(a, b)\)

如何求 \(ax + by' = c\) 呢？

\(ax + by' = c\)
\(\Rightarrow (\frac{gcd(a, b)}{c})(ax + by') = gcd(a, b)\)
\(\Rightarrow ax' + by'' = gcd(a, b)\)

使用扩展欧几里德算法求出来的 \(x'\) 不是 \(x\)，需要 \(x = x' \times \frac{c}{gcd(a, b)}\)

特殊的

定理2：若 \(gcd(a, b) = 1\)，且 \(x_0, y_0\) 为方程 \(ax + by = c\) 的一组解，则该方程的任意可表示为：\(x = x_0 + bt, y = y_0 + at\) 且对任意整数 \(t\) 都成立。

根据定理 2，可以求出方程的所有解。但在实际问题中，我们往往被要求求出一个最小整数解，也就是一个特解 \(x = (x\ mod\ t + t)\ mod\ t\)，其中 \(t = \frac{b}{gcd(a, b)}\)
