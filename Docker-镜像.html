commit 镜像

docker commit
提交容器成为一个新的副本
  • docker commit -m="提交的描述信息" -a="作者" 容器id 目标容器名:[TAG]
  • docker commit -a="nl" -m="add webapps app" tomcat01 tomcat01:1.0
  • docker images：查看创建的镜像，发现有 tomcat01:1.0

发布镜像到 dockerhub

  • https://hub.docker.com/
  • docker login
  • docker login -u ogmc
  • 因为 docker 官方镜像仓库是以用户名来命名仓库的，所以需要修改镜像名字
    • docker tag tomcat-jdk ogmc/tomcat-jdk:1.0
  • docker push
  • docker pull
