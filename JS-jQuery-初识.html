DOM 对象和 jQuery 对象

  • DOM 对象转换为 jQuery 对象

    var $obj= $("#id");   // jQuery 对象  
var dom = $obj[0];    // DOM 对象
alert(dom.checked)    // 检测这个 checkbox 是否被选中

    var $obj = $("#id");     // jQuery 对象
var dom = $obj.get(0);   // DOM 对象
alert(dom.checked);      // 检测这个 checkbox 是否被选中

  • jQuery 对象转换为 DOM 对象

    var dom = document.getElementById("v");   // DOM 对象 
var $obj = $(dom);   // jQuery 对象
