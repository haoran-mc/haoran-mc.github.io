牛顿迭代法

> 牛顿迭代法求平方根
> 由于牛顿迭代法有局限性，所以不能将所有类型的函数根求出，也不能求出函数的所有根，但是对于平方根，牛顿迭代法求出其根
> [牛顿迭代法](https://blog.csdn.net/ccnt_2012/article/details/81837154)

#include <cstdio>
const double eps = 1e-6;

double mySqrt(double base) {
    double x = 1.0;
    double y;
    while(x*x-base < -eps || x*x-base > eps) {
        y = (x + base/x) / 2.0;
        x = y;
    }
    return x;
}

int main() {
    printf("%f\n", mySqrt(82.0));
    return 0;
}
