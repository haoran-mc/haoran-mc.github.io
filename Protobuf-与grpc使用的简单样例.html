gRPC + protobuf 使用的简单样例

文件目录：

project
 ├── go.mode
 ├── go.sum
 ├── proto
 │   ├── helloworld.proto
 │   └── build.sh
 ├── server
 │   └── main.go
 └── client
     └── main.go

代码：

  • go.mod
  • proto/helloworld.proto
  • proto/build.sh
  • server/main.go
  • client/main.go

  1. 下载安装 grpc、protobuf

    go install google.golang.org/protobuf/cmd/protoc-gen-go@v1.28
go install google.golang.org/grpc/cmd/protoc-gen-go-grpc@v1.2
    • Debian下载protoc

  2. 使用 proto 文件生成 go 代码

    cd proto
./build.sh

    build.sh

    #!/bin/bash

protoc --go_out=. --go_opt=paths=source_relative --go-grpc_out=. --go-grpc_opt=paths=source_relative ./helloworld.proto
    –go_out
    指定生成的 go 文件路径
    –go-grpc_out
    指定生成的 grpc 文件路径
    ./helloworld.proto
    是指定使用的 proto 文件

    生成 go 代码后的目录：

    project
├── go.mode
├── go.sum
├── proto
│   ├── helloworld.pb.go
│   ├── helloworld.proto
│   ├── helloworld_grpc.pb.go
│   └── build.sh
├── server
│   └── main.go
└── client
    └── main.go
  3. 编写服务端代码

    1. 取出 server

      type server struct {
    proto.UnimplementedHelloServer
}

    2. 挂载方法

      func (s *server) SayHello(ctx context.Context, req *proto.Request) (res *proto.Response, err error) {
    fmt.Println(req.GetMsg())
    return &proto.Response{Msg: "hello world"}, nil
}

    3. 注册服务

      l, _ := net.Listen("tcp", ":8888")
s := grpc.NewServer()
proto.RegisterHelloServer(s, &server{})

    4. 创建监听

      s.Serve(l)
  4. 编写客户端代码

    1. 创建一个链接

      conn, _ := grpc.Dial("localhost:8888", grpc.WithTransportCredentials(insecure.NewCredentials()))

    2. new一个client

      client := proto.NewHelloClient(conn)

    3. 调用client的方法

      req, _ := client.SayHello(context.Background(), &proto.Request{Msg: "client here"})

    4. 获取返回值

      fmt.Println(req.GetMsg())
