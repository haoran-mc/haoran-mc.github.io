翻转字符串

问题描述

请实现一个算法，在不使用【额外数据结构和储存空间】的情况下，翻转一个给定的字符串(可以使用单个过程变量)。

给定一个string，请返回一个string，为翻转后的字符串。保证字符串的长度小于等于5000。

解题思路

翻转字符串其实是将一个字符串以中间字符为轴，前后翻转，即将 str[len] 赋值给 str[0] ，将 str[0] 赋值 str[len] 。

代码参考

package main

import "fmt"

func reverString(str string) (string, bool) {
        var revstr []rune = ([]rune)(str)
        l := len(revstr)
        if l > 5000 {
                return str, false
        }
        for i := 0; i < l / 2; i ++ {
                revstr[i], revstr[l - 1 - i] = revstr[l - 1 - i], revstr[i]
        }
        return string(revstr), true
}

func main() {
        var str string = "Hello, world!"
        fmt.Println(str)

        if revstr, ok := reverString(str); ok {
                fmt.Println(revstr)
        } else {
                fmt.Println("error")
        }
}

代码解析

以字符串长度的1/2为轴，前后赋值。
