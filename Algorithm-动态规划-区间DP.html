最长回文字串

  • dp[i][j]表示str[i]至str[j]所表示的子串是否是回文子串，是则为1，不是为0，根据str[i]是否等于str[j]
    • 若str[i] == str[j]，那么只要str[i+1]至str[j-1]是回文子串，str[i]至str[j]就是回文子串，如果str[i+1]至str[j-1]不是回文子串，则str[i]至str[j]也不是回文子串
    • 若str[i] != str[j]，那么str[i]至str[j]一定不是回文子串
    • 动态转移方程：
      • dp[i][j] = dp[i+1][j-1] (str[i] == str[j])
      • dp[i][j] = 0 (str[i] != str[j])
      • 边界dp[i][i] = 1; dp[i][i+1] = (str[i] == str[j])?1:0
      • 但是如果按照i和j从小到大的顺序来枚举子串的两个端点，然后更新dp[i][j]，会无法保证dp[i+1][j-1]已经被计算过，从而无法得到正确的dp[i][j]
    • 所以这种方法不行，观察看到转移方程按子串的长度逐渐减少，所以考虑按子串的长度和子串的初始位置进行枚举
