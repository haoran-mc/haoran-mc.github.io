js

var div = document.createElement("div");
div.id = "mydiv";
div.style.font = "bold 14px" ;
div.color = "green";
div.innerHTML = "Javascript DIV";
document.body.appendChild(div);

jQuery

简单使用

$("<div>", {
    id: "mydiv",
    text: "Javascript DIV",
}).css({
    "font-size": "14px" ,
    "font-weight": "bold",
    "color": "green",
}).appendTo("body");

更多方法

html
给元素添加 html 代码或者清空 html 代码（参数为空字符串）；
append
向元素的末尾添加 html 代码；
appendTo
这个方法跟 append 方法的很像，只是要添加的 html 代码的目标有所不一样；
prepend
在被选元素的开头插入内容
after
向元素的后边添加 html 代码，如果元素后面有元素了，那将后面的元素后移，然后将html代码插入；
before
向元素的前边添加 html 代码，如果元素前面有元素了，那将前面的元素前移，然后将html代码插入；
insertAfter
将 JQuery 封装好的元素插入到指定元素的后面，如果元素后面有元素了，那将后面的元素后移，然后将 JQuery 对象插入；
insertBefore
将 JQuery 封装好的元素插入到指定元素的前面，如果元素前面有元素了，那将前面的元素前移，然后将 JQuery 对象插入；
empty
清空元素内部的 html 代码，它只是清空内部的 html 代码，但是标记仍然留在 DOM 中；
remove
从 DOM 中移除整个元素；

一方面， insertAfter 是将封装好的元素插到指定位置后面；另一方面 insertAfter 也可以理解为 moveAfter，将已有的元素插入到指定位置之后，比如 $('#table-of-contents').insertAfter('postamble'); ； after 是添加 html 代码。

建议首选 insertAfter 。

function afterText() {
    var txt1 = "<b>I </b>";                    // Create element with HTML 
    var txt2 = $("<i></i>").text("love ");     // Create with jQuery
    var txt3 = document.createElement("big");  // Create with DOM
    txt3.innerHTML = "jQuery!";
    $("img").after(txt1, txt2, txt3);          // Insert new elements after img

    $('div').insertAfter('img');               // Insert div after img
}

.wrap()

1. 传标签
$('.block').addClass('active').wrap('<div class="wrapper"/>');
2. 传函数
$('.block').addClass('active').wrap(function (index) {
    return '<div class="wrapper' + index + '"></div>'
})
3. 传对象
$('.block').addClass('active').wrap($('.wrapper'));

.wrapAll()

$('.block').wrapAll('<div class="wrapper"/>');

在所有匹配元素外面包一层 HTML 结构。

.wrapInner()

$('.block').wrapInner('<div class="new"></div>');

在匹配元素里的内容外包一层结构。

.unwrap()

将匹配元素集合的父级元素删除，保留自身（和兄弟元素，如果存在）在原来的位置，不接受参数。
